3. **การตัดสิน CSV vs API เปลี่ยนโปรไฟล์ความเสี่ยงทั้งใบ**: ถ้า **CSV path ชนะ** → SC-001 เสี่ยงน้อยลงมาก (ส่งไฟล์เดียว ไม่ใช่ 10000 calls) แต่ **PDPA เสี่ยงขึ้น** (ไฟล์ PII ที่ rest → TC-074 กลายเป็น P0 จริง). ถ้า **API path ชนะ** → PDPA-at-rest หายไป แต่ **SC-001 แทบแน่นอนว่า Fail**. ต้อง re-run `risk` (iteration +1) หลังมีคำตัดสิน
