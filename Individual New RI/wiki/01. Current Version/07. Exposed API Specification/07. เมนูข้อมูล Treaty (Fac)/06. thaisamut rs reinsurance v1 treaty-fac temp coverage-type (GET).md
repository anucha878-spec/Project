## Overview

สำหรับข้อมูล Coverage Type ด้วย reinsurer code และ treaty code

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/treaty-fac/temp/coverage-type

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Input

| Field Name | Type | Description |
| --- | --- | --- |
| cfReinsurerCode | String | Reinsurer Code ของตาราง cf_reinsurer |
| cfTreatyCode | String | Treaty Code ของตาราง cf_treaty |

## Process

1. ค้นหาข้อมูล Coverage Type จากตาราง cf_coverage_type โดยทำการ join cf_reinsure และ cf_treaty ที่ status = A เพื่อนำ cf_treaty_id มาใช้งาน
2. ให้ return ค่าเป็น List<String> ของ coverage type ที่เจอในตาราง cf_coverage_type
