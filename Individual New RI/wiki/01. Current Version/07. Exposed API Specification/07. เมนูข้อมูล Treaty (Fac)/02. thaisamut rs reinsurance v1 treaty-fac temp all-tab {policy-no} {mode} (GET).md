## Overview

สำหรับดึงข้อมูลหน้า View กรมธรรม์ Facultative

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/treaty-fac/temp/all-tab/{policy-no}/{mode}

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Input

<แสดงข้อมูล Parameter ที่ต้องการ>

| Field | Type | Description | Example |
| --- | --- | --- | --- |
| policyNo | String | เลขที่กรมธรรม์ | 12032452 |
| mode | ModeEnum | Mode ของการดึงข้อมูล | APRV, VIEW |

## Process

1. กรณีที่ mode = APRV ให้ดึงข้อมูลจำพวกตาราง temp ทั้งหมด
2. กรณีที่ mode = VIEW ให้ดึงข้อมูลจำพวกตาราง cf ทั้งหมด
