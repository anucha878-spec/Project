## Process

1. บันทึกข้อมูลการเพิ่มหรือแก้ไขที่กรอกไว้ [บันทึกข้อมูลลง Table : tx_tmp_policy_fac_hd](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1113588271) (อัพเดทฟิลด์ effective_date_from, effective_date_to ที่มีการแก้ไข)[บันทึกข้อมูลลง Table : tx_tmp_policy_fac_dt](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1113588295) โดยแยกตาม Mode ของข้อมูลที่ส่งมา Add, Edit, Delete
2. กรณีที่ข้อมูล [tx_tmp_policy_fac_hd](/display/RDSINRI/tx_tmp_policy_fac_hd).ms_process_status != WAIT ให้อัพเดทเป็น WAIT (หากเป็น WAIT อยู่แล้วไม่ต้องทำอะไร)
3. กรณีที่ข้อมูล [tx_tmp_policy_fac_hd](/display/RDSINRI/tx_tmp_policy_fac_hd).ms_process_status จากข้อ 2. มีการอัพเดท (เปลี่ยนสถานะ) ให้ทำการบันทึกข้อมูล [tx_policy_history](/display/RDSINRI/tx_policy_history) โดยบันทึก his_action เป็น ค่าเดียวกับข้อ 2. หากไม่มีการอัพเดทสถานะไม่ต้องบันทึกข้อมูลเพิ่ม
