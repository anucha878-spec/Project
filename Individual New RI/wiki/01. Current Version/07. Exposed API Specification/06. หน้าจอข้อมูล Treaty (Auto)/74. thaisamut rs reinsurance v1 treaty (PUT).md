## Overview

สำหรับแก้ไขข้อมูลหน้าจอข้อมูล Treaty (Auto)

**Repositories**: msa-reinsurance

**Service path**

**PUT**:****/thaisamut/rs/reinsurance/v1/treaty

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete
