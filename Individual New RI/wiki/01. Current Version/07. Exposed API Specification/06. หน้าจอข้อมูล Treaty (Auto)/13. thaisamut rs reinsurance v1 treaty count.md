## Overview

สำหรับหน้าจอ IRI_CFG_001 หน้าจอข้อมูล Treaty (Auto) : นับจำนวนข้อมูล Treaty ที่มีอยู่ในระบบตาม criteria

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/treaty/count

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Input

<แสดงข้อมูล Parameter ที่ต้องการ>

| Name | Type | Data Length | Require | Description | Example | Validation |
| --- | --- | --- | --- | --- | --- | --- |
| reinsurerCode | string |  | N | Reinsurer Code |  |  |
| treatyCode | string |  | N | Treaty Code |  |  |
| effectiveDateFrom | string |  | N | Effective Date From |  |  |
| effectiveDateTo | string |  | N | Effective Date To |  |  |
