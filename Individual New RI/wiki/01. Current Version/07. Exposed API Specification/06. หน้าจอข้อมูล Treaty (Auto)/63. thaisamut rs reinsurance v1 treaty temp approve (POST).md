Overview

สำหรับพิจารณา Treaty (Mode Approve)

**Repositories**: msa-reinsurance

**Service path**

**POST**:****/thaisamut/rs/reinsurance/v1/treaty/temp/approve

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Input

<แสดงข้อมูล Parameter ที่ต้องการ>

| Name | Type | Require | Description | Example |
| --- | --- | --- | --- | --- |
| treatyId | Long | Y | id ของตาราง tx_tmp_treaty |  |
| action | String | Y | Lookup ที่ [cf_lookup_catalog](http://wiki.thaisamut.co.th/display/RDSINRI/cf_lookup_catalog).parent_id = 1000100 | EDIT, APRV |
| remark | String | Y | หมายเหตุ | ทดสอบ |

## Process

1. อัพเดทตาราง tx_tmp_treaty.ms_process_status ตาม action ที่เลือกมา
2. insert ข้อมูลลงตาราง tx_treaty_history
3. นำข้อมูลจากตาราง tx_tmp ทั้งหมดมาบันทึกที่ตาราง cf เป็นการขนข้อมูลเพื่อมาบันทึกที่ master
