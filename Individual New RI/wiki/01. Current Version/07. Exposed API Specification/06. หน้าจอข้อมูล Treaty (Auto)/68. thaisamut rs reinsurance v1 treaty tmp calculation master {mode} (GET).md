## Overview

สำหรับดึงข้อมูลรายละเอียด Master Calculation จากตาราง ms_calculation และ ms_formula

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/treaty/tmp/calculation/master/{mode}

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Input

<แสดงข้อมูล Parameter ที่ต้องการ>

| Name | Type | Require | Description | Example |
| --- | --- | --- | --- | --- |
| mode | Enum | Y | mode ของการกรองข้อมูลฟิลด์ ms_calculation.ms_cal_case | YEARLY,MONTHLY |

## Process

1. ค้นหาข้อมูลที่ตาราง ms_calculation และ ms_formular ด้วย mode เพื่อนำไปใช้สำหรับการ default ค่า auto complete ที่หน้าจอข้อมูล Treaty (Auto) > Tab 7 Calculation

## Output

<แสดงข้อมูลที่จะได้รับจาก service นี้>

| Name | Type | Example |
| --- | --- | --- |
| responseCode | String | 000 |
| responseMessage | String | SUCCESS |

## Exception

<อธิบายว่า มี exception อะไรที่ต้องจัดการหรือระวังบ้าง>
