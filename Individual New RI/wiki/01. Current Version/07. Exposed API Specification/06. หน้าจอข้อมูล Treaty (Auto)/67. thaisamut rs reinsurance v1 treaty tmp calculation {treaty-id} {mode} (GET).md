## Overview

สำหรับดึงข้อมูลรายละเอียด Calculation จากตาราง tx_tmp_calculation

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/treaty/tmp/calculation/{treaty-id}/{mode}

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Input

<แสดงข้อมูล Parameter ที่ต้องการ>

| Name | Type | Require | Description | Example |
| --- | --- | --- | --- | --- |
| mode | Enum | Y | mode ของการกรองข้อมูลฟิลด์ ms_calculation.ms_cal_case | YEARLY,MONTHLY |
| treatyId |  | Y | id ของตาราง tx_tmp_treaty |  |

## Process

1. ค้นหาข้อมูลที่ตาราง tx_tmp_treaty_cal ด้วย mode และ treatyId เพื่อนำข้อมูลในเป็นค่า default ของหน้าจอ Calculation

## Output

<แสดงข้อมูลที่จะได้รับจาก service นี้>

| Name | Type | Example |
| --- | --- | --- |
| responseCode | String | 000 |
| responseMessage | String | SUCCESS |
