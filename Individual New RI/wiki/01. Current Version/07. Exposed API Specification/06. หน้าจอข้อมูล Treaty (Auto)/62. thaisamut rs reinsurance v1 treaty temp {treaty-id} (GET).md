## Overview

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/treaty/temp/{treaty-id}

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete
