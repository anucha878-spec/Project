## Overview

สำหรับตรวจสอบว่า Treaty นี้บันทึกไปถึง Tab ไหนแล้ว ทั้งหมด 7 Tab เพื่อใช้ในการ disable ปุ่มแต่ละ Tab

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/treaty/temp/check-curent-tab/{treaty-id}

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Input

<แสดงข้อมูล Parameter ที่ต้องการ>

| Name | Type | Require | Description | Example |
| --- | --- | --- | --- | --- |
| treatyId | String | Y | id ของตาราง tmp_treaty |  |

## Process

ตรวจสอบข้อมูลแต่ละ Tab ว่ามีข้อมูลหรือไม่ โดยใช้สำหรับการ disable ปุ่ม Tab ซึ่งให้ return ค่าเป็น currentTab ออกมาว่า Tab ที่มากที่สุดที่มีข้อมูลคือ Tab ไหน โดยค้นหาข้อมูลด้วย treatyId
