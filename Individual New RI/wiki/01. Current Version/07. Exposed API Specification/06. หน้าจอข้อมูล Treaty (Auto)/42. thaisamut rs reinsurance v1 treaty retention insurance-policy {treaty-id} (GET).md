Overview

สำหรับดึงข้อมูลหน้าจอ Treaty (Auto) หน้าย่อย Setup Retention (Tab4) (Insurance Policies)

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/treaty/retention/insurance-policy

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Input

<แสดงข้อมูล Parameter ที่ต้องการ>

| Name | Type | Require | Description | Example |
| --- | --- | --- | --- | --- |
| treatyId | Long | Y | id ของตาราง tx_tmp_treaty | 123 |
| msPolicyType | String | Y | Ms Policy Type |  |

## Process

1. ดึงข้อมูลออกจากตาราง tx_tmp_insurance_policy โดยการ join กับตาราง tx_tmp_retention และใช้เงื่อนไข treatyId, MsPolicyType ในการค้นหาSELECT ip.retention_name , ip.ms_policy_type as insurance_policies, ip.retention_amount ,   ip.effective_date_from , ip.effective_date_to , ip.updated_date, ip.tmp_insurance_policy_id   FROM tx_tmp_insurance_policy ip   INNER JOIN tx_tmp_retention r   ON ip.tmp_retention_id = r.tmp_retention_id   WHERE r.ms_medical_type = :msPolicyType   AND r.tmp_treaty_id = :treatyId
