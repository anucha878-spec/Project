Overview

สำหรับดึงข้อมูล PlanCode จากตาราง tx_tmp_plancode_hd มาแสดงที่หน้าจอ Setup RI Premium > Setup by product

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/treaty/premium/setup-product/{treaty-id}

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Input

<แสดงข้อมูล Parameter ที่ต้องการ>

| Name | Type | Require | Description | Example |
| --- | --- | --- | --- | --- |
| treatyId | Long | Y | id ของตาราง tx_tmp_treaty | 123 |

## Process

1. ดึงข้อมูลออกจากตาราง tx_tmp_plancode_hd และ tx_tmp_prem_by_product_hd โดยใช้เงื่อนไข treatyId
