## Overview

สำหรับแก้ไขข้อมูลหน้าจอข้อมูล Reinsurer

**Repositories**: msa-reinsurance

**Service path**

**PUT**:****/thaisamut/rs/reinsurance/v1/reinsurer

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Process

1. **ตรวจสอบข้อมูลก่อนว่าครบตามที่ระบบต้องการหรือไม่ กรณีที่ไม่ถูกต้องให้ตอบกลับ**

| Name | Type | Example |
| --- | --- | --- |
| responseCode | String | 100 |
| responseMessage | String | Invalid Request |

**2. update ข้อมูลตาราง cf_reinsurer**

| DB Table | DB Field | Source (request) |
| --- | --- | --- |
| cf_reinsurer | reinsurer_name | reinsurerName |
| cf_reinsurer | country | country |
| cf_reinsurer | address | address |
| cf_reinsurer | telephone | telephone |
| cf_reinsurer | fax | fax |
| cf_reinsurer | email | email |
| cf_reinsurer | website | website |
| cf_reinsurer | effective_date_from | effective_date_from |
| cf_reinsurer | effective_date_to | effective_date_to |
| cf_reinsurer | updated_by | userLogin() |
| cf_reinsurer | updated_date | now() |

*** กรณีที่ input request มาเป็น null ให้ update เป็น db เป็น null ด้วย

**3. update ข้อมูลตาราง cf_reinsurer_contact**

3.1 .ให้ดูค่า field action ถ้า

- None ไม่ต้องทำอะไร

- Add ให้เพิ่มข้อมูลเพิ่ม

- Edit ให้แก้ไขข้อมูล

- Delete ให้ลบข้อมูล

| DB Table | DB Field | Source (request) |
| --- | --- | --- |
| cf_reinsurer_contact | contact_name | contactName |
| cf_reinsurer_contact | contact_position | contactPosition |
| cf_reinsurer_contact | contact_email | contactEmail |
| cf_reinsurer_contact | contact_telephone | contactTelephone |
| cf_reinsurer_contact | contact_fax | contactFax |
| cf_reinsurer_contact | updated_date | now() |
| cf_reinsurer_contact | updated_by | userLogin() |
