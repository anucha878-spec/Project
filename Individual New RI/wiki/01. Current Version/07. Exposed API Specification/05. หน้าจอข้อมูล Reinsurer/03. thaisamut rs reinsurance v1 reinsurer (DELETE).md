## Overview

สำหรับลบข้อมูล reinsurer + ข้อมูล contact

**Repositories**: msa-reinsurance

**Service path**

**DELETE**:****/thaisamut/rs/reinsurance/v1/reinsurer

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Input

<แสดงข้อมูล Parameter ที่ต้องการ>

| Name | Type | Require | Description | Example |
| --- | --- | --- | --- | --- |
| id | Long | Y |  | 1 |

## Process

1. ตรวจสอบตรวจสอบขข้อมูลก่อนลบ รายการที่ผู้ใช้งานเลือก กับ cf_treaty.cf_reinsurer_id โดยต้องไม่มีอยู่ในทุกรายการของ Table cf_treatyถึงแม้ว่ารายการนั้นจะมี Effective Date To ผ่านไปแล้ว ก็ไม่สามารถลบได้
2. ค้นหาข้อมูลที่ตาราง cf_reinsurer_contact โดยค้นหาโดย cf_reinsurer_id ถ้าพบให้ทำการลบข้อมูล
3. ลบข้อมูลที่ตาราง cf_reinsurer
