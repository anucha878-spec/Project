## Overview

สำหรับตรวจสอบข้อมูล Treaty ว่ามีการผูกอยู่กับ Reinsure ที่ต้องการจะลบหรือไม่ (เคสลบข้อมูล reinsurer)

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/treaty/reinsurer-active

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

| reinsurerCode | string | Y | Reinsurer Code |  |  |
|  |  |  |  |  |  |

## Process

1. ค้นหาข้อมูลจากตาราง cf_reinsurer โดย reinsurer_code

2. ตรวจสอบว่ามีข้อมูล cf_reinsurer.cf_reinsurer_id ในตาราง cf_treaty หรือไม่

- ถ้าพบว่ามีข้อมูลให้

| Name | Type | Description | Example |
| --- | --- | --- | --- |
| responseCode | String |  | 000 |
| responseMessage | String |  | SUCCESS |
| reinsurerActive | boolean |  | true |

- ถ้าไม่พบข้อมูลให้

| Name | Type | Description | Example |
| --- | --- | --- | --- |
| responseCode | String |  | 000 |
| responseMessage | String |  | SUCCESS |
| reinsurerActive | boolean |  | false |
