## Overview

สำหรับหน้าจอ IRI_CFG_004 หน้าจอข้อมูล Profit Commission: นับจำนวนข้อมูล Profit Commission ที่มีอยู่ในระบบตาม criteria

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/profit-comm/count

****

Icon
TYPE : <GET, POST, PUT, DELETE>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

## Process

นับจำนวนข้อมูลจากตาราง cf_profit_comm, cf_profit_comm_treaty โดยค้นหาข้อมูลที่มี cf_profit_comm.status = A เท่านั้น

และรองรับการค้นหาด้วย limit และ offset

| Name | Type | Description | Example |
| --- | --- | --- | --- |
