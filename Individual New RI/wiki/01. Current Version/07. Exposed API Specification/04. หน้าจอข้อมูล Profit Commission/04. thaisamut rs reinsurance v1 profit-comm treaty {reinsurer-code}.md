## Overview

**Repositories**: msa-reinsurance

**Service path**

**GET**:****/thaisamut/rs/reinsurance/v1/profit-comm/treaty/{reinsurer-code}

****

Icon
TYPE : <GET, POST, PUT, DELETE>

2. return ค่าดังนี้Field NameTypeRemark**List**  cf_treaty_idLong cf_reinsurer_idLong reinsurer_codeString treaty_codeString treaty_nameString
