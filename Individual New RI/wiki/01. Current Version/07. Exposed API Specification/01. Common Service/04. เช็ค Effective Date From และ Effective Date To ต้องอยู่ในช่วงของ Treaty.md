Overview

ใช้สำหรับการเช็ค Effective Date From และ Effective Date To ของ Retention ต้องอยู่ในช่วงของ Effective Date From และ Effective Date To ของ Treaty

**Repositories**: msa-reinsurance

**Service Name** : TreatyCompoent.validateTreatyEffectiveDate()

## Parameter

| Name | Type | Description |
| --- | --- | --- |
| tmpTreatyId | Long | id ของตาราง tx_tmp_treaty |
| effectiveDateFrom | Date | ค่าของ Effective Date From ที่ต้องการจะเช็ค |
| effectiveDateTo | Date | ค่าของ Effective Date To ที่ต้องการจะเช็ค |

## Process

นำ effectiveDateFrom และ effectiveDateTo ไปค้นหาในตาราง tx_tmp_treaty ด้วยเงื่อนไข tmpTreatyId โดยตรวจสอบว่า ค่าของ effectiveDateFrom และ effectiveDateTo อยู่ในช่วงของ treaty หรือไม่

กรณีที่อยู่ในช่วง ให้ระบบ return ค่า true

กรณีที่ไม่อยู่ในช่วง ให้ระบบ return ค่า false

## Output

| Type | Example | Description |
| --- | --- | --- |
| boolean | true | กรณีที่อยู่ในช่วง ให้ระบบ return ค่า true \| กรณีที่ไม่อยู่ในช่วง ให้ระบบ return ค่า false |

## Exception

<อธิบายว่า มี exception อะไรที่ต้องจัดการหรือระวังบ้าง>
