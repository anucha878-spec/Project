ส่วน Detail
**Step1 :** นำ parameter: {TEMPLATE_ID} ไปเทียบกับ [cf_soa_template](http://wiki.thaisamut.co.th/display/RDSINRI/cf_soa_template).cf_template_file_id

**Step2 :** ค้นหา [cf_soa_template](http://wiki.thaisamut.co.th/display/RDSINRI/cf_soa_template).cf_soa_column_id ตามรายละเอียดดังนี้

***หมายเหตุ:***

- *ข้อมูลที่แสดงเป็นระดับ Quarter แต่ข้อมูลใน Table [tx_ri_act_soa](http://wiki.thaisamut.co.th/display/RDSINRI/41.+tx_ri_act_soa) จะเป็นระดับรอบ period จึงต้องมีการ sum ค่า field นั้นๆ โดย group by [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd)*.quarter_period, quarter_year ก่อนนำมาแสดง**
- **กรณีข้อมูลเป็น 0 หรือ NULL ให้แสดงเป็น "-"**
