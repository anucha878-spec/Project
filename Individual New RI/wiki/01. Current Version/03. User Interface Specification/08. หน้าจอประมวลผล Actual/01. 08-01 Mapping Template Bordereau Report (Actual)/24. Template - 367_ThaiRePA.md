## การออกรายงาน

การออกรายงาน
- Bordereau Report จะสามารถออกรายงานได้จาก [หน้าจอประมวลผล Actual](/pages/viewpage.action?pageId=1098613351)
- ข้อมูลที่ใช้ในรายงานมีรายละเอียดดังนี้ข้อมูล File ดูจาก [cf_template_file](/display/RDSINRI/cf_template_file)ข้อมูล Sheet ดูจาก [cf_template_sheet](/display/RDSINRI/cf_template_sheet)ข้อมูล Field ดูจาก [cf_template_field](/display/RDSINRI/cf_template_field)
- ข้อมูลในส่วน Report Detailข้อมูลการประมวลผลหลักจาก [tx_ri_act_hd](/display/RDSINRI/25.+tx_ri_act_hd) โดยรายงานจะออกเป็นราย quarterข้อมูลการประมวลผลหลักจาก [tx_ri_act_dt](/display/RDSINRI/43.+tx_ri_act_dt) โดยรายงานจะออกเป็นราย period  โดย 1 quarter จะมี 3 period เพื่อออกรายงาน ([tx_ri_act_hd](/display/RDSINRI/25.+tx_ri_act_hd) 1 รายการจะมี tx_ri_act_dt 3 รายการ)ข้อมูลกรมธรรม์จาก [tx_ri_act_policy_dt](/display/RDSINRI/26.+tx_ri_act_policy_dt)ข้อมูลลูกค้าจาก [tx_ri_est_cust_dt](/display/RDSINRI/15.+tx_ri_est_cust_dt)ข้อมูล Claim จาก [tx_ri_act_claim_dt](/display/RDSINRI/29.+tx_ri_act_claim_dt)ข้อมูลการคำนวณข้อมูลการคำนวณ Sheet New,Renew : [tx_ri_bdr_act_new_renew](/display/RDSINRI/34.+tx_ri_bdr_act_new_renew)ข้อมูลการคำนวณ Sheet Alteration : [tx_ri_bdr_act_alter](/display/RDSINRI/35.+tx_ri_bdr_act_alter)ข้อมูลการคำนวณ Sheet Claim : [tx_ri_bdr_act_claim](/display/RDSINRI/36.+tx_ri_bdr_act_claim)

1. ข้อมูลการประมวลผลหลักจาก [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd) ด้วยเงื่อนไขquarter_period รอบประมวลแบบ quarter จาก [หน้าจอประมวลผล Actual](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1098613351) โดย 1 quarter จะประกอบไปด้วย period 3 รายการ เพื่อนำข้อมูลมาแสดงใน 3 Sheet(Claim)cf_treaty_id id ของ treaty ที่เลือกจาก [หน้าจอประมวลผล Actual](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1098613351)facult_flag (ข้อมูล Facultative หรือ Auto) ที่เลือกจาก [หน้าจอประมวลผล Actual](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1098613351)
2. ข้อมูลที่ได้จากหน้าจอ จากข้อ 1 จะได้ [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd) .ri_act_hd_id
3. ข้อมูลperiod รอบประมวลแบบรายเดือน ([tx_ri_act_dt](http://wiki.thaisamut.co.th/display/RDSINRI/43.+tx_ri_act_dt).period) ด้วยเงื่อนไข [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd) .ri_act_hd_id = [tx_ri_act_dt](http://wiki.thaisamut.co.th/display/RDSINRI/43.+tx_ri_act_dt).ri_act_hd_id (inner join)
4. ข้อมูลกรมธรรม์ ด้วยเงื่อนไข [tx_ri_bdr_act_claim](http://wiki.thaisamut.co.th/display/RDSINRI/36.+tx_ri_bdr_act_claim)[.ri_act_policy_dt_id](http://wiki.thaisamut.co.th/display/RDSINRI/26.+tx_ri_act_policy_dt) =  [tx_ri_act_policy_dt](http://wiki.thaisamut.co.th/display/RDSINRI/26.+tx_ri_act_policy_dt).ri_act_policy_dt_id (inner join)
5. ข้อมูลลูกค้าด้วยเงื่อนไข ด้วยเงื่อนไข [tx_ri_bdr_act_claim](http://wiki.thaisamut.co.th/display/RDSINRI/36.+tx_ri_bdr_act_claim).ri_act_cust_dt_id = [tx_ri_est_cust_dt](http://wiki.thaisamut.co.th/display/RDSINRI/15.+tx_ri_est_cust_dt).ri_est_cust_dt_id (inner join)
6. เงื่อนไขการดึงข้อมูล [tx_ri_bdr_act_claim](http://wiki.thaisamut.co.th/display/RDSINRI/36.+tx_ri_bdr_act_claim) . ri_act_dt_id = id ข้อมูล period จากข้อ3
7. Order by [tx_ri_bdr_claim](http://wiki.thaisamut.co.th/display/RDSINRI/24.+tx_ri_bdr_claim).claim_no ASC
