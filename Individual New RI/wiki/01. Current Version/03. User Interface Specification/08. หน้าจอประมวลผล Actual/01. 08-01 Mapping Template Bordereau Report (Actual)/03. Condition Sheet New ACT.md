เงื่อนไขการดึงข้อมูล Sheet : New
1. ข้อมูลการประมวลผลหลักจาก [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd) ด้วยเงื่อนไขquarter_period  รอบประมวลแบบ quarter จาก [หน้าจอประมวลผล Actual](/pages/viewpage.action?pageId=1098613351) โดย 1 quarter จะประกอบไปด้วย period 3 รายการ เพื่อนำข้อมูลมาแสดงใน 3 Sheet(New)cf_treaty_id id ของ treaty ที่เลือกจาก [หน้าจอประมวลผล Actual](/pages/viewpage.action?pageId=1098613351)facult_flag (ข้อมูล Facultative หรือ Auto) ที่เลือกจาก [หน้าจอประมวลผล Actual](/pages/viewpage.action?pageId=1098613351)
2. ข้อมูลที่ได้จากหน้าจอ จากข้อ 1 จะได้ [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd) .ri_act_hd_id
3. ข้อมูลperiod  รอบประมวลแบบรายเดือน ([tx_ri_act_dt](http://wiki.thaisamut.co.th/display/RDSINRI/43.+tx_ri_act_dt).period) ด้วยเงื่อนไข [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd) .ri_act_hd_id = [tx_ri_act_dt](/display/RDSINRI/43.+tx_ri_act_dt).ri_act_hd_id (inner join)
4. ข้อมูลกรมธรรม์ ด้วยเงื่อนไข [tx_ri_bdr_act_new_renew](http://wiki.thaisamut.co.th/display/RDSINRI/34.+tx_ri_bdr_act_new_renew)[.ri_act_policy_dt_id](http://wiki.thaisamut.co.th/display/RDSINRI/26.+tx_ri_act_policy_dt) =  [tx_ri_act_policy_dt](http://wiki.thaisamut.co.th/display/RDSINRI/26.+tx_ri_act_policy_dt).ri_act_policy_dt_id (inner join)
5. ข้อมูลลูกค้าด้วยเงื่อนไข ด้วยเงื่อนไข [tx_ri_bdr_act_new_renew](http://wiki.thaisamut.co.th/display/RDSINRI/34.+tx_ri_bdr_act_new_renew) .ri_act_cust_dt_id = [tx_ri_est_cust_dt](/display/RDSINRI/15.+tx_ri_est_cust_dt).ri_est_cust_dt_id (inner join)
6. เงื่อนไขการดึงข้อมูล Sheet : New  [tx_ri_bdr_act_new_renew](http://wiki.thaisamut.co.th/display/RDSINRI/34.+tx_ri_bdr_act_new_renew).report_type = 'NEW'  และ [tx_ri_bdr_act_new_renew](http://wiki.thaisamut.co.th/display/RDSINRI/34.+tx_ri_bdr_act_new_renew). ri_act_dt_id = id ข้อมูล period จากข้อ 3
7. Order by [tx_ri_bdr_new_renew](http://wiki.thaisamut.co.th/display/RDSINRI/22.+tx_ri_bdr_new_renew).policy_no ASC
