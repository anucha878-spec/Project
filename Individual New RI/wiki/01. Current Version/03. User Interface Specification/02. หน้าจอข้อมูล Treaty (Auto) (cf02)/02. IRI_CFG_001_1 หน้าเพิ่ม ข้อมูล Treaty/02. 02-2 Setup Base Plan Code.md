### Assumption

- ผู้ใช้งานต้อง Log in เข้าใช้งานอย่างถูกต้อง
- ผู้ใช้งานต้องมีสิทธิ์การเข้าถึงหน้าจอ การบันทึก/แก้ไข รายการ Treaty

### Screen Overview

- เพื่อ การบันทึก/แก้ไข รายการ Treaty

### Pre-Condition

1. ผู้ใช้งานต้องมีสิทธิ์การใช้งานของ Maker และมองเห็นปุ่มเพิ่ม Treaty ในหน้า [IRI_CFG_001 หน้ารวม + ค้นหา Treaty (cf02)](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1098613387)
2. ผู้ใช้งานต้องกรอกข้อมูลในหน้า [02-1 ข้อมูลหลัก Treaty](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1100350120) เรียบร้อยแล้วจึงจะแสดงหน้าจอนี้
