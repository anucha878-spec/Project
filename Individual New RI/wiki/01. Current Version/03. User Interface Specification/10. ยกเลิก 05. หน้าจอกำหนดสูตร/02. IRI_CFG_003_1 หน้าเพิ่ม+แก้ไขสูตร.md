**Description**

| No. | หัวข้อ | รายละเอียด |
| --- | --- | --- |
| **1** | **วัตถุประสงค์ (Objective)** | เพิ่ม / แก้ไข สูตรของค่าคำนวน |
| **2** | **ผู้ใช้งาน และ สิทธิ์ที่ได้รับ (Target Users and authorization)** | เจ้าหน้าที่ฝ่ายคณิตศาสตร์ |
| **3** | **ข้อมูลที่ป้อนเข้าสู่ระบบ** | **สำหรับเงื่อนไขการค้นหาข้อมูล**เลือก ค่าคำนวน จาก Master ข้อมูลที่กำหนดไว้เลือก กรณี ที่เกี่ยวข้องกับค่าคำนวนเลือก การนับเศษวัน ที่เกี่ยวข้องกับค่าคำนวนกรอกหมายเหตุ (ถ้ามี)สร้างสูตรโดยการเลือกตัวแปรตามกรอกสีฟ้าปุ่มล้างข้อมูลปุ่มลบสูตรในกล่องกำหนดสูตรปุ่มยกเลิกปุ่มบันทึก |
| **4** | **ข้อมูลที่ได้จากระบบ (Output)** | Label สำหรับกรอกข้อมูลเกี่ยวกับการกำหนดสูตรทั้งหมด |
| **5** | **การกระทำกับหน้าจอ (Actions)** | เลือกรายการค่าคำนวนและกรอกข้อมูลสูตรตามที่ระบบกำหนดกดปุ่มยกเลิกกดปุ่มบันทึก |
| **6** | **อธิบายรายละเอียด (Description)** | **Field****แสดงข้อมูลและเงื่อนไขการตรวจสอบข้อมูล****ค่าคำนวน**บังคับเลือกแสดงตัวเลือกในรูปแบบ Dropdownตัวเลือกจะเป็นข้อมูล Master ค่าคำนวนที่กำหนดไว้**กรณี**บังคับเลือกแสดงตัวเลือกในรูปแบบ Dropdownกรุณาเลือก (Default)ไม่กำหนดLifeRiderPAรายเดือนรายปีApproveDecline**การนับเศษวัน**บังคับเลือกแสดงตัวเลือกในรูปแบบ Dropdownกรุณาเลือก (Default)ไม่กำหนดนับเป็นรายวันนับเป็น 1 เดือน**หมายเหตุ**กรอกเป็นตัวหนังสือหรือตัวเลขเท่านั้น**สูตร**สามารถคลิกวาง Cursor ตามที่ต้องการได้ไม่สามารถพิมพ์ข้อความได้**ปุ่มล้างข้อมูล**เมื่อกดปุ่ม สูตรในช่องสูตรจะถูกล้างออกไปทั้งหมด**ตัวแปรต่าง ๆ**รายการกล่องข้อความที่ระบุตัวแปรเมื่อกด ข้อความตัวแปรบนกล่องจะแสดงบนช่องสูตร**ปุ่มลบ**เมื่อกด ตัวแปรที่อยู่ในช่องสูตร จะหายไปทีละ 1 ตัวแปร จากด้านขวาสุด **ปุ่ม ยกเลิก**ยกเลิกการเพิ่มหรือแก้ไขข้อมูลที่กรอกไว้**ปุ่ม บันทึก**ตรวจสอบว่ามีการกรอกข้อมูลครบทุกช่องบันทึกข้อมูลการเพิ่มหรือแก้ไขที่กรอกไว้ในรูปแบบ Draft ทั้งหมด และปิด Popup |
| **Field** | **แสดงข้อมูลและเงื่อนไขการตรวจสอบข้อมูล** |
| **ค่าคำนวน** | บังคับเลือกแสดงตัวเลือกในรูปแบบ Dropdownตัวเลือกจะเป็นข้อมูล Master ค่าคำนวนที่กำหนดไว้ |
| **กรณี** | บังคับเลือกแสดงตัวเลือกในรูปแบบ Dropdownกรุณาเลือก (Default)ไม่กำหนดLifeRiderPAรายเดือนรายปีApproveDecline |
| **การนับเศษวัน** | บังคับเลือกแสดงตัวเลือกในรูปแบบ Dropdownกรุณาเลือก (Default)ไม่กำหนดนับเป็นรายวันนับเป็น 1 เดือน |
| **หมายเหตุ** | กรอกเป็นตัวหนังสือหรือตัวเลขเท่านั้น |
| **สูตร** | สามารถคลิกวาง Cursor ตามที่ต้องการได้ไม่สามารถพิมพ์ข้อความได้ |
| **ปุ่มล้างข้อมูล** | เมื่อกดปุ่ม สูตรในช่องสูตรจะถูกล้างออกไปทั้งหมด |
| **ตัวแปรต่าง ๆ** | รายการกล่องข้อความที่ระบุตัวแปรเมื่อกด ข้อความตัวแปรบนกล่องจะแสดงบนช่องสูตร |
| **ปุ่มลบ** | เมื่อกด ตัวแปรที่อยู่ในช่องสูตร จะหายไปทีละ 1 ตัวแปร จากด้านขวาสุด |
