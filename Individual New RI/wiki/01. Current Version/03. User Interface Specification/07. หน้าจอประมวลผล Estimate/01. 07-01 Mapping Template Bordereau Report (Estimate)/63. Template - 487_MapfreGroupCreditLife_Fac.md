- ข้อมูลในส่วน Report Detailข้อมูลการประมวลผลหลักจาก [tx_ri_est_hd](/display/RDSINRI/13.+tx_ri_est_hd)ดึงข้อมูลกรมธรรม์จาก [tx_ri_est_policy_dt](/display/RDSINRI/14.+tx_ri_est_policy_dt)ดึงข้อมูลลูกค้าจาก [tx_ri_est_cust_dt](/display/RDSINRI/15.+tx_ri_est_cust_dt)ดึงข้อมูล Claim จาก [tx_ri_est_claim_dt](/display/RDSINRI/17.+tx_ri_est_claim_dt)ดึงข้อมูลการคำนวณจากข้อมูลการคำนวณ Sheet New,Renew : [22. tx_ri_bdr_new_renew](/display/RDSINRI/22.+tx_ri_bdr_new_renew)ข้อมูลการคำนวณ Sheet Alteration : [23. tx_ri_bdr_alter](/display/RDSINRI/23.+tx_ri_bdr_alter)ข้อมูลการคำนวณ Sheet Claim : [24. tx_ri_bdr_claim](/display/RDSINRI/24.+tx_ri_bdr_claim)

## ตารางคำอธิบาย

Template รายงาน
ตารางคำอธิบาย sheet New
เงื่อนไขการดึงข้อมูล Sheet : New
1. ข้อมูลการประมวลผลหลักจาก [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd) ด้วยเงื่อนไขperiod รอบประมวลที่เลือกจาก จาก [หน้าจอประมวลผล Estimate](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1098613343)cf_treaty_id id ของ treaty ที่เลือกจาก [หน้าจอประมวลผล Estimate](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1098613343)facult_flag (ข้อมูล Facultative หรือ Auto) ที่เลือกจาก [หน้าจอประมวลผล Estimate](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1098613343)
2. ข้อมูลที่ได้จากหน้าจอจากข้อ 1 จะได้ [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd).ri_est_hd_id
3. ข้อมูลกรมธรรม์ด้วยเงื่อนไข  [tx_ri_bdr_new_renew](/display/RDSINRI/22.+tx_ri_bdr_new_renew).ri_est_policy_dt_id = [tx_ri_est_policy_dt](/display/RDSINRI/14.+tx_ri_est_policy_dt).ri_est_policy_dt_id (inner join)
4. ข้อมูลลูกค้าด้วยเงื่อนไข [tx_ri_bdr_new_renew](/display/RDSINRI/22.+tx_ri_bdr_new_renew).ri_est_cust_dt_id = [tx_ri_est_cust_dt](/display/RDSINRI/15.+tx_ri_est_cust_dt).ri_est_cust_dt_id (inner join)
5. เงื่อนไขการดึงข้อมูล Sheet : New [tx_ri_bdr_new_renew](/display/RDSINRI/22.+tx_ri_bdr_new_renew).report_type = 'NEW' และ [tx_ri_bdr_new_renew](/display/RDSINRI/22.+tx_ri_bdr_new_renew).ri_est_hd_id = id ที่ได้จากหน้าจอ
6. Order by [tx_ri_bdr_new_renew](/display/RDSINRI/22.+tx_ri_bdr_new_renew).policy_no ASC
