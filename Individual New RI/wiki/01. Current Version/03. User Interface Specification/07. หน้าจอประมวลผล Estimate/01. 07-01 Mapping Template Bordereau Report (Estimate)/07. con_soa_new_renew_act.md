เงื่อนไขการดึงข้อมูล ACT
1. ข้อมูลการประมวลผลหลักจาก [tx_ri_act_hd](/display/RDSINRI/25.+tx_ri_act_hd) ด้วยเงื่อนไข ri_est_hd_id เป็น Key join เงื่อนไขที่ใช้ facult_flag (ข้อมูล Facultative หรือ Auto)
2. ข้อมูลกรมธรรม์ด้วยเงื่อนไข [tx_ri_act_base_auto_dt](/display/RDSINRI/30.+tx_ri_act_base_auto_dt).ri_act_policy_dt_id = [tx_ri_act_policy_dt](/display/RDSINRI/26.+tx_ri_act_policy_dt).ri_act_policy_dt_idด้วยเงื่อนไข [tx_ri_act_rider_auto_dt](/display/RDSINRI/31.+tx_ri_act_rider_auto_dt).ri_act_policy_dt_id = [tx_ri_act_policy_dt](/display/RDSINRI/26.+tx_ri_act_policy_dt).ri_act_policy_dt_idด้วยเงื่อนไข [tx_ri_act_base_facult_dt](/display/RDSINRI/32.+tx_ri_act_base_facult_dt).ri_act_policy_dt_id = [tx_ri_act_policy_dt](/display/RDSINRI/26.+tx_ri_act_policy_dt).ri_act_policy_dt_idด้วยเงื่อนไข [tx_ri_act_rider_facult_dt](/display/RDSINRI/33.+tx_ri_act_rider_facult_dt).ri_act_policy_dt_id = [tx_ri_act_policy_dt](/display/RDSINRI/26.+tx_ri_act_policy_dt).ri_act_policy_dt_id เงื่อนไขที่ใช้ ปีกรมธรรม์ BASE ([tx_ri_act_policy_dt](/display/RDSINRI/26.+tx_ri_act_policy_dt).policy_year) เงื่อนไขที่ใช้ ปีกรมธรรม์ RIDER ([tx_ri_act_rider_auto_dt](/display/RDSINRI/31.+tx_ri_act_rider_auto_dt).policy_year_rider) หรือ ([tx_ri_act_rider_facult_dt](/display/RDSINRI/33.+tx_ri_act_rider_facult_dt) .policy_year_rider)
3. ข้อมูลการคำนวณ : NEW, RENEWด้วยเงื่อนไข [tx_ri_act_base_auto_dt](/display/RDSINRI/30.+tx_ri_act_base_auto_dt).report_type เป็น (NEW, RENEW) ด้วยเงื่อนไข [tx_ri_act_rider_auto_dt](/display/RDSINRI/31.+tx_ri_act_rider_auto_dt).report_type เป็น (NEW, RENEW)ด้วยเงื่อนไข [tx_ri_act_base_facult_dt](/display/RDSINRI/32.+tx_ri_act_base_facult_dt).report_type เป็น (NEW, RENEW)ด้วยเงื่อนไข [tx_ri_act_rider_facult_dt](/display/RDSINRI/33.+tx_ri_act_rider_facult_dt).report_type เป็น (NEW, RENEW)
4. ข้อมูลการคำนวณ : ALTERด้วยเงื่อนไข [tx_ri_act_base_auto_dt](/display/RDSINRI/30.+tx_ri_act_base_auto_dt).report_type เป็น (ALTER) ด้วยเงื่อนไข [tx_ri_act_rider_auto_dt](/display/RDSINRI/31.+tx_ri_act_rider_auto_dt).report_type เป็น (ALTER)ด้วยเงื่อนไข [tx_ri_act_base_facult_dt](/display/RDSINRI/32.+tx_ri_act_base_facult_dt).report_type เป็น (ALTER)ด้วยเงื่อนไข [tx_ri_act_rider_facult_dt](/display/RDSINRI/33.+tx_ri_act_rider_facult_dt).report_type เป็น (ALTER)
5. ข้อมูลการคำนวณ : CLAIMด้วยเงื่อนไข [tx_ri_act_base_auto_dt](/display/RDSINRI/30.+tx_ri_act_base_auto_dt).report_type เป็น (CLAIM) ด้วยเงื่อนไข [tx_ri_act_rider_auto_dt](/display/RDSINRI/31.+tx_ri_act_rider_auto_dt).report_type เป็น (CLAIM)ด้วยเงื่อนไข [tx_ri_act_base_facult_dt](/display/RDSINRI/32.+tx_ri_act_base_facult_dt).report_type เป็น (CLAIM)ด้วยเงื่อนไข [tx_ri_act_rider_facult_dt](/display/RDSINRI/33.+tx_ri_act_rider_facult_dt).report_type เป็น (CLAIM)
