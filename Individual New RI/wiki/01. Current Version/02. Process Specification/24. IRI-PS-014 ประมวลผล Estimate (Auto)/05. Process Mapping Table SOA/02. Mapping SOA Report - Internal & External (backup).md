Mapping ข้อมูล Auto
เงื่อนไขการดึงข้อมูล EST
1. ข้อมูลการประมวลผลหลักจาก [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd) ด้วยเงื่อนไข ri_est_hd_id เป็น Key join เงื่อนไขที่ใช้ facult_flag (ข้อมูล Facultative หรือ Auto)
2. ข้อมูลกรมธรรม์ด้วยเงื่อนไข [tx_ri_est_base_auto_dt](/display/RDSINRI/18.+tx_ri_est_base_auto_dt).ri_est_policy_dt_id = [tx_ri_est_policy_dt](/display/RDSINRI/14.+tx_ri_est_policy_dt).ri_est_policy_dt_idด้วยเงื่อนไข [tx_ri_est_rider_auto_dt](/display/RDSINRI/19.+tx_ri_est_rider_auto_dt).ri_est_policy_dt_id = [tx_ri_est_policy_dt](/display/RDSINRI/14.+tx_ri_est_policy_dt).ri_est_policy_dt_idด้วยเงื่อนไข [tx_ri_est_base_facult_dt](/display/RDSINRI/20.+tx_ri_est_base_facult_dt).ri_est_policy_dt_id = [tx_ri_est_policy_dt](/display/RDSINRI/14.+tx_ri_est_policy_dt).ri_est_policy_dt_idด้วยเงื่อนไข [tx_ri_est_rider_facult_dt](/display/RDSINRI/21.+tx_ri_est_rider_facult_dt).ri_est_policy_dt_id = [tx_ri_est_policy_dt](/display/RDSINRI/14.+tx_ri_est_policy_dt).ri_est_policy_dt_id เงื่อนไขที่ใช้ ปีกรมธรรม์ BASE ([tx_ri_est_policy_dt](/display/RDSINRI/14.+tx_ri_est_policy_dt).policy_year)เงื่อนไขที่ใช้ ปีกรมธรรม์ RIDER ([tx_ri_est_rider_auto_dt](/display/RDSINRI/19.+tx_ri_est_rider_auto_dt).policy_year_rider) หรือ ([tx_ri_est_rider_facult_dt](/display/RDSINRI/21.+tx_ri_est_rider_facult_dt).policy_year_rider)
3. ข้อมูลการคำนวณ : NEW, RENEWด้วยเงื่อนไข [tx_ri_est_base_auto_dt](/display/RDSINRI/18.+tx_ri_est_base_auto_dt).report_type เป็น (NEW, RENEW) ด้วยเงื่อนไข [tx_ri_est_rider_auto_dt](/display/RDSINRI/19.+tx_ri_est_rider_auto_dt).report_type เป็น (NEW, RENEW)ด้วยเงื่อนไข [tx_ri_est_base_facult_dt](/display/RDSINRI/20.+tx_ri_est_base_facult_dt).report_type เป็น (NEW, RENEW)ด้วยเงื่อนไข [tx_ri_est_rider_facult_dt](/display/RDSINRI/21.+tx_ri_est_rider_facult_dt).report_type เป็น (NEW, RENEW)
4. ข้อมูลการคำนวณ : ALTERด้วยเงื่อนไข [tx_ri_est_base_auto_dt](/display/RDSINRI/18.+tx_ri_est_base_auto_dt).report_type เป็น (ALTER) ด้วยเงื่อนไข [tx_ri_est_rider_auto_dt](/display/RDSINRI/19.+tx_ri_est_rider_auto_dt).report_type เป็น (ALTER)ด้วยเงื่อนไข [tx_ri_est_base_facult_dt](/display/RDSINRI/20.+tx_ri_est_base_facult_dt).report_type เป็น (ALTER)ด้วยเงื่อนไข [tx_ri_est_rider_facult_dt](/display/RDSINRI/21.+tx_ri_est_rider_facult_dt).report_type เป็น (ALTER)
5. ข้อมูลการคำนวณ : CLAIMด้วยเงื่อนไข [tx_ri_est_base_auto_dt](/display/RDSINRI/18.+tx_ri_est_base_auto_dt).report_type เป็น (CLAIM) ด้วยเงื่อนไข [tx_ri_est_rider_auto_dt](/display/RDSINRI/19.+tx_ri_est_rider_auto_dt).report_type เป็น (CLAIM)ด้วยเงื่อนไข [tx_ri_est_base_facult_dt](/display/RDSINRI/20.+tx_ri_est_base_facult_dt).report_type เป็น (CLAIM)ด้วยเงื่อนไข [tx_ri_est_rider_facult_dt](/display/RDSINRI/21.+tx_ri_est_rider_facult_dt).report_type เป็น (CLAIM)
