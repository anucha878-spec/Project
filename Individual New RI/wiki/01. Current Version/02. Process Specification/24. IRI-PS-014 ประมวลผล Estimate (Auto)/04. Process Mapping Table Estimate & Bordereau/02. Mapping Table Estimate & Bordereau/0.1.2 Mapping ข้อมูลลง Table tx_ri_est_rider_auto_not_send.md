0.1.2 Mapping ข้อมูลลง Table tx_ri_est_rider_auto_not_send
เงื่อนไขการ Mapping ข้อมูลลง Table [tx_ri_est_rider_auto_not_send](http://wiki.thaisamut.co.th/x/D4CaRw)

***หมายเหตุ:** บันทึกเฉพาะรายการที่ rider_code ไม่เท่ากับ 0*

*กรณีที่เป็นกรมธรรม์ประเภท PA ให้สร้าง record RIDER 1 record โดยนำ plan_code มาเก็บไว้ใน rider_code*

- *บันทึกเฉพาะกรมที่ประมวลผลแต่เมื่อผ่านขั้นตอนที่ [12 ขั้นตอนการเลือก Policy ส่งประกันต่อ](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1131446319) แล้วไม่ต้องส่งประกันต่อ.*
- *กรมที่มีรายการ Claim ที่มีสถานะ NOCAL_NOSEND_REPO*
- *กรมที่มีรายการ Claim ที่ไม่คำนวณ ต้องนำข้อมูลอื่น ๆ มาเก็บไว้เพื่อนำไปประมวลผลค่าใช้จ่ายได้***
