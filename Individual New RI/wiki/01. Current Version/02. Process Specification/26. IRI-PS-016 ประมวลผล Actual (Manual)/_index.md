ทำการดึงข้อมูลรายการ Alteration โดยตรวจสอบรายการตามขั้นตอนดังนี้

1. นำ parameter: {QUATER_YEAR}, {QUATER_PERIOD} มาแปลงจาก Quarter เป็นรอบ period ตามเงื่อนไขดังนี้Q1 แปลงเป็น เดือน 1-3Q2 แปลงเป็น เดือน 4-6Q3 แปลงเป็น เดือน 7-9Q4 แปลงเป็น เดือน 10-12**ตัวอย่างเช่น** 2023/Q4 แปลงเป็น 3 period คือ 202310, 202311, 202312
2. ทำการดึงข้อมูลรายการ Alteration ทีละ period ตามขั้นตอนดังนี้นำ period ที่ได้จากข้อ 1. มาแปลงเป็นช่วงวันที่ ตัวอย่างเช่น period 202310 = 01/10/2023 - 31/10/2023ดึงข้อมูล Alteration ด้วยเงื่อนไขดังนี้ค้นหารายการ Alteration ที่ วันที่ Alter (suthanee.sa 04/09/2025) ตรวจสอบระหว่าง [tx_ri_est_alt_dt](http://wiki.thaisamut.co.th/display/RDSINRI/16.+tx_ri_est_alt_dt).alter_date กับ [tx_ri_est_alt_dt](http://wiki.thaisamut.co.th/display/RDSINRI/16.+tx_ri_est_alt_dt).alter_update_date รายการใดมีค่ามากกว่า ให้ใช้ค่านั้น (suthanee.sa 04/09/2025)นำรายการที่ได้ไปเทียบให้อยู่ในช่วงวันที่ ที่ได้จากข้อ a. เมื่อได้รายการ Alteration แล้ว ให้นำ [tx_ri_est_alt_dt](http://wiki.thaisamut.co.th/display/RDSINRI/16.+tx_ri_est_alt_dt).ri_est_policy_dt_id ไปค้นหาที่ [tx_ri_est_policy_dt](http://wiki.thaisamut.co.th/display/RDSINRI/14.+tx_ri_est_policy_dt).ri_est_policy_dt_id จะได้ [tx_ri_est_policy_dt](http://wiki.thaisamut.co.th/display/RDSINRI/14.+tx_ri_est_policy_dt).ri_est_hd_idนำ [tx_ri_est_policy_dt](http://wiki.thaisamut.co.th/display/RDSINRI/14.+tx_ri_est_policy_dt).ri_est_hd_id ไปค้นหาที่ [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd).id โดยมีเงื่อนไขดังนี้[tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd).treaty_code ต้องตรงกับ parameter: {TREATY_CODE}[tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd).reinsurer_code ต้องตรงกับ parameter: {REINSURER_CODE}[tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd).status = 'APPROVE'เมื่อได้รายการ [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd).id ที่ตรงตามเงื่อนไขแล้ว ให้นำรายการที่ได้ มาอยู่ใน period นั้นๆ และบันทึกข้อมูลลง Table Actual ตามรายละเอียด [Process Mapping Table Actual & Bordereau](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1132757529) ส่วนข้อที่ 1.
