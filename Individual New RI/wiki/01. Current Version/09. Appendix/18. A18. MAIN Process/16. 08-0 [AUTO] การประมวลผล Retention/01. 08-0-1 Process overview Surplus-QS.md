2.คำนวณ OLI 1st Layer, OLI 2nd Layer, Max Retention Layer 1 (คงเหลือ) ,Max Retention Layer 2 (คงเหลือ)

- กรณีเป็นการคำนวณกรมธรรม์แรกคำนวณ OLI 1st Layer = Max Retention Layer 1 ลบ Current NARคำนวณ Max Retention Layer 1 (คงเหลือ)กรณี Max Retention Layer 1 มากกว่า Current NAR : Max Retention Layer 1 (คงเหลือ) = OLI 1st Layerกรณี Max Retention Layer 1 น้อยกว่าหรือเท่ากับ Current NAR : Max Retention Layer 1 (คงเหลือ) = 0.00 และคำนวณ Current NAR(คงเหลือ) = Current NAR ลบ OLI 1st Layerคำนวณ OLI 2nd Layer และ Max Retention Layer 2 (คงเหลือ) กรณี Max Retention Layer 1 (คงเหลือ) มากกว่า 0.00 ไม่ต้องคำนวณ OLI 2nd Layer และ Max Retention Layer 2 (คงเหลือ) กรณี Max Retention Layer 1 (คงเหลือ) เท่ากับ 0.00 และ Current NAR(คงเหลือ) มากกว่า 0.00คำนวณ OLI 2nd Layer = Current NAR(คงเหลือ) คูณ %Layer2กรณี OLI 2nd Layer มากกว่าหรือเท่ากับ Max Retention Layer 2 ให้ Max Retention Layer 2 (คงเหลือ) = 0.00 และ  OLI 2nd Layer = Max Retention Layer 2กรณี OLI 2nd Layer น้อยกว่า Max Retention Layer 2 ให้ Max Retention Layer 2 (คงเหลือ) = Max Retention Layer 2
- กรณีเป็นการคำนวณกรมธรรม์ถัดไปกรณี Max Retention Layer 1 (คงเหลือ) มากกว่า 0.00คำนวณ OLI 1st Layer = Max Retention Layer 1 (คงเหลือ) ลบ Current NARคำนวณ Max Retention Layer 1 (คงเหลือ)กรณี Max Retention Layer 1 (คงเหลือ) มากกว่า Current NAR : Max Retention Layer 1 (คงเหลือ) = OLI 1st Layerกรณี Max Retention Layer 1 (คงเหลือ) น้อยกว่าหรือเท่ากับ Current NAR : Max Retention Layer 1 (คงเหลือ) = 0.00 และคำนวณ Current NAR(คงเหลือ) = Current NAR ลบ OLI 1st Layerคำนวณ OLI 2nd Layer และ Max Retention Layer 2 (คงเหลือ) กรณี Max Retention Layer 1 (คงเหลือ) มากกว่า 0.00 ไม่ต้องคำนวณ OLI 2nd Layer และ Max Retention Layer 2 (คงเหลือ) กรณี Max Retention Layer 1 (คงเหลือ) เท่ากับ 0.00 และ Current NAR(คงเหลือ) มากกว่า 0.00คำนวณ OLI 2nd Layer = Current NAR(คงเหลือ) คูณ %Layer2กรณี OLI 2nd Layer มากกว่าหรือเท่ากับ Max Retention Layer 2 ให้ Max Retention Layer 2 (คงเหลือ) = 0.00 และ  OLI 2nd Layer = Max Retention Layer 2กรณี OLI 2nd Layer น้อยกว่า Max Retention Layer 2 ให้ Max Retention Layer 2 (คงเหลือ) = Max Retention Layer 2กรณี Max Retention Layer 1 (คงเหลือ) เท่ากับ 0.00 และ Max Retention Layer 2 (คงเหลือ) มากกว่า 0.00คำนวณ OLI 2nd Layer = Current NAR(คงเหลือ) คูณ %Layer2กรณี OLI 2nd Layer มากกว่าหรือเท่ากับ Max Retention Layer 2 ให้ Max Retention Layer 2 (คงเหลือ) = 0.00 และ OLI 2nd Layer = Max Retention Layer 2กรณี OLI 2nd Layer น้อยกว่า Max Retention Layer 2 ให้ Max Retention Layer 2 (คงเหลือ) = Max Retention Layer 2กรณี Max Retention Layer 2 (คงเหลือ) เท่ากับ 0.00 จะไม่คำนวณ OLI 2nd Layer เนื่องจากถือว่าเป็น Retestion เต็มแล้ว จึงจะส่ง ประกันภัยต่อ 100%

3.คำนวณ SR

- กรณี Max Retention Layer 1 (คงเหลือ) มากกว่า 0.00 ไม่ต้องคำนวณ SR (ค่าเป็น 0.00)
- กรณี Max Retention Layer 1 (คงเหลือ) เท่ากับ 0.00 และ Max Retention Layer 2 (คงเหลือ) มากกว่า 0.00SR = Max Retention Layer 1 บวก OLI 2nd Layer
- กรณี Max Retention Layer 2 (คงเหลือ) = 0.00 และ OLI 2nd Layer มากกว่า 0.00 Max Retention Layer 1 บวก (Max Retention Layer 2 คูณ %Layer2)
- กรณีไม่เข้าเงื่อนไขให้ SR = 0.00
