กรณีรายเดือน
- กรณี วันที่เริ่มต้นสัญญาของ Rider ตรงกับวันที่เริ่มต้นสัญญาของ BASE สามารถใช้เงื่อนไขเหมือน BASE
- กรณี วันที่เริ่มต้นสัญญาของ Rider ไม่ตรงกับ วันที่เริ่มต้นสัญญาของ BASE ในการส่ง RE ครั้งแรก (New ปีที่ 1 เดือนที่ 1) ถ้า treaty_code ที่ประมวลผลอยู่ในกลุ่ม cf_lookup_catalog.parent_id <> 1005400ระบบต้องสร้างรายการให้มีรอบตรงกับ BASEถ้า treaty_code ที่ประมวลผลอยู่ในกลุ่ม cf_lookup_catalog.parent_id = 1005400ให้ประมวลผลตามเงื่อนไขด้านล่าง

กรณีรายปี

- ถ้าเดือนของ commencement_date เท่ากับ rider_com_date จะใช้ข้อมูล วัน/เดือน เหมือน base (รอบความคุ้มครองตรงกัน เดือนวันที่เริ่มสัญญาของ base และ rider ตรงกัน)
- ถ้าเดือนของ commencement_date ไม่เท่ากับ rider_com_date จะใช้ ข้อมูล วัน/เดือน ของ rider_com_date แทน
