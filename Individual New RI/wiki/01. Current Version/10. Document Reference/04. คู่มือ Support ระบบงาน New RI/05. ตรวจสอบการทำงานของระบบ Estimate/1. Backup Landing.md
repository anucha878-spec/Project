3. หน้าจอจะแสดง Batch ทั้งหมดของระบบ

4. สามารถตรวจสอบได้ โดยการนำชื่อ Batch ที่ได้จากข้อ 1. Batch ที่เกี่ยวข้อง [ms_process](http://wiki.thaisamut.co.th/display/RDSINRI/11.+ms_process).process_code และตรวจสอบ "วันที่ Run Batch" รวมกับ "สถานะ" ว่า Batch มีการทำงานได้ปกติใช่หรือไม่ (วันที่และเวลาทำงานให้ตรวจสอบจาก [05. ตรวจสอบการทำงานของระบบ](http://wiki.thaisamut.co.th/x/TATgT) โดย Batch จะทำงานพร้อมกันทีละ 2 รายการ ตาม คิวใน Table [ms_process](http://wiki.thaisamut.co.th/display/RDSINRI/11.+ms_process).seq_landing
