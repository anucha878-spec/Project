## Overview

สำหรับ Update สถานะและ count จำนวนรายการที่ระบบ EDW

**Repositories**: msa-reinsurance

**Service path**

**PUT**:******thaisamut/rs/reinsurance/v1/process-estimate/status**

****

Icon
TYPE : <GET>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Process

ดำเนินการ Return ข้อมูล [tx_ri_process_header](http://wiki.thaisamut.co.th/display/RDSADW/tx_ri_process_header) กลับไปที่ [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd) ดังนี้

1. เมื่อระบบดำเนินการปรับสถานะข้อมูลที่ [tx_ri_process_header](http://wiki.thaisamut.co.th/display/RDSADW/tx_ri_process_header) (msa-adwetl) ให้ get ข้อมูลสถานะรายการและผู้ดำเนินการ กลับมา update ที่ [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd) ดังนี้Return ri_process_status_id จาก [tx_ri_process_header](http://wiki.thaisamut.co.th/display/RDSADW/tx_ri_process_header).ri_process_status_id ไป Update ที่ [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd).edw_statusReturn process_status_name จาก [ms_ri_process_status](http://wiki.thaisamut.co.th/display/RDSADW/ms_ri_process_status).process_status_name ไป Update ที่ [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd).edw_status_descReturn update_by จาก [tx_ri_process_header](http://wiki.thaisamut.co.th/display/RDSADW/tx_ri_process_header).update_by ไป Update ที่ [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd).updated_by
