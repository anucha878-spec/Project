## Overview

สำหรับส่งข้อมูลประมวลผล Estimate เข้าระบบ EDW

**Repositories**: msa-claimtx

**Service path**

**GET**:****/thaisamut/rs/claimtx/v1/reinsurance/claim-cb/{from}/{size}

****

Icon
TYPE : <POST>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Process

ดำเนินการ Insert ข้อมูลเข้า EDW ผ่าน Process โดยให้ trigger เมื่อกดปุ่ม ยืนยันข้อมูล EST.ที่ หน้าจอ [08. หน้าจอประมวลผล Actual](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1098613351)

- ดำเนินการสร้าง queue ในการส่งข้อมูล โดยตรวจสอบจากรายการที่กดยืนยันข้อมูลจากหน้าจอ
- กรณีเลือก ยืนยันข้อมูล มากกว่า 1 รายการให้ทำรายการแรกให้สำเร็จก่อน แล้วค่อยทำ queue ต่อไป ตามขั้นตอนดังนี้
