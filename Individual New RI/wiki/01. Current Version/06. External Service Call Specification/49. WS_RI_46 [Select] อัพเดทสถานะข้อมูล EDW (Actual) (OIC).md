## Overview

สำหรับ Update สถานะและ count จำนวนรายการที่ระบบ EDW (OIC)

**Repositories**: msa-reinsurance

**Service path**

****PUT: thaisamut/rs/reinsurance/v1/process-estimate/status

****

Icon
TYPE : <GET>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete

## Process

ดำเนินการ Return ข้อมูล [tx_ri_process_header_oic](http://wiki.thaisamut.co.th/display/RDSADW/tx_ri_process_header_oic) กลับไปที่ [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd)ดังนี้

1. เมื่อระบบดำเนินการปรับสถานะข้อมูลที่ [tx_ri_process_header_oic](http://wiki.thaisamut.co.th/display/RDSADW/tx_ri_process_header_oic) (msa-adwetl) ให้ get ข้อมูลสถานะรายการและผู้ดำเนินการ กลับมา update ที่ [tx_ri_est_hd](http://wiki.thaisamut.co.th/display/RDSINRI/13.+tx_ri_est_hd) ดังนี้Return ri_process_status_id จาก [tx_ri_process_header_oic](http://wiki.thaisamut.co.th/display/RDSADW/tx_ri_process_header_oic).ri_process_status_id ไป Update ที่ [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd).edw_statusReturn process_status_name จาก [ms_ri_process_status](http://wiki.thaisamut.co.th/display/RDSADW/ms_ri_process_status).process_status_name ไป Update ที่ [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd).edw_status_descReturn ri_process_status_id จาก [tx_ri_process_header_oic](http://wiki.thaisamut.co.th/display/RDSADW/tx_ri_process_header_oic).ms_import_status_id ไป Update ที่ [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd).mps_statusReturn process_status_name จาก [ms_mps_import_status](http://wiki.thaisamut.co.th/display/RDSADW/ms_mps_import_status).import_status_name ไป Update ที่ [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd).mps_status_descReturn update_by จาก [tx_ri_process_header_oic](http://wiki.thaisamut.co.th/display/RDSADW/tx_ri_process_header_oic).update_by ไป Update ที่ [tx_ri_act_hd](http://wiki.thaisamut.co.th/display/RDSINRI/25.+tx_ri_act_hd).updated_by
