## Overview

สำหรับดึงข้อมูลยอดสุทธิเพื่อเปรียบเทียบรายการจาก 2 ระบบคือ IRI และ EDW (Estimate)

**Repositories**: msa-reinsurance

**Service path**

**GET**:******thaisamut/rs/reinsurance/v1/process-estimate/standard-hd/{ri-std-hd-id}**

****

Icon
TYPE : <GET>

**อธิบายได้ดังนี้**

GET - Select

POST - Insert

PUT - Update

DELETE - Delete
