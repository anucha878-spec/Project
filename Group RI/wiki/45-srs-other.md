===== PAGE 1289389357 | Software Requirements Specification > 01. Introduction =====
Background (Project Description)
- ปัจจุบันบริษัทมีสัญญาประกันภัยต่อ ประเภทกลุ่ม กับ 3 บริษัท
- ในแต่ละสัญญามีเงื่อนไขการคำนวณที่แตกต่างกัน โดยมีกรมธรรม์ที่ต้องจัดส่งตามสัญญาประกันภัยต่อประมาณ 620 กรมธรรม์ มีจำนวนสมาชิกรวม 250,000 คนต่อปี ซึ่งปริมาณข้อมูลเพิ่มขึ้นเรื่อยๆ
- ต้องใช้เวลาในตรวจสอบและจัดทำรายงาน 40 วันต่อไตรมาส จากวันทำงานทั้งหมด 65 วันต่อไตรมาส โดยปัจจุบันคำนวณในรูปแบบ Excel file ทั้ง Estimated report และ Actual report
- มีผู้ปฏิบัติงานที่รับผิดชอบ 1 คนสำหรับจัดทำ และ 1 คนสำหรับตรวจสอบ
Purpose
- พัฒนาระบบให้ Automate การจัดทำรายงานที่เกี่ยวข้องกับการประกันภัยต่อสำหรับ Group Business เพื่อลดขั้นตอนในการทำงานและโอกาสการเกิดความผิดพลาด
- ลดเวลาในการจัดทำรายงาน Reinsurance Bordereaux สำหรับข้อมูลประมาณการรายเดือนและ Actual รายไตรมาส
- เชื่อมต่อข้อมูลเข้าสู่ EDW และ IFRS17 และรองรับรายงานสำหรับการส่ง คปภ.
Scope
| Reinsurance Process | Reinsurance Report | Enterprise Data Warehouse (EDW) |
| เชื่อมต่อและดึงข้อมูลจากระบบประกันกลุ่มและข้อมูลสินไหมเข้าสู่ระบบ New Group RIคำนวณทุนประกันต่อ เบี้ยประกันต่อ สินไหมประกันต่อ แยกตามแต่ละสัญญา ทั้งหมด 3 สัญญาคำนวณได้ทั้งการประมาณการรายเดือน และข้อมูล Actual สำหรับส่งประกันภัยต่อ | ตรวจสอบความถูกต้องและความครบถ้วนของข้อมูล RI ก่อนการสร้างรายงานGenerate Reinsurance Bordereaux ตาม Template ของแต่ละสัญญาฯ และจัดเก็บในรูปแบบมาตรฐานจัดเก็บและค้นหาข้อมูลการส่งประกันต่อตามช่วงเวลาที่ต้องการ | ส่งข้อมูลประกันต่อเข้าสู่ EDW อัตโนมัติ ในระดับรายกรมธรรม์จัดเตรียมข้อมูล RI ให้พร้อมใช้งาน สำหรับฝ่ายคณิตศาสตร์, ฝ่ายบัญชี และฝ่ายการเงิน |

## Performance Requirements
- สามารถสืบค้นข้อมูลจากระบบได้ในระยะเวลาไม่เกิน 10 วินาที
- สามารถบันทึกข้อมูลแล้วเสร็จได้ในระยะเวลาไม่เกิน 3 วินาที

## Availability/Continuity Requirements
- สามารถให้บริการได้ตลอดเวลาในช่วงวันทำการของบริษัท
- หากระบบมีปัญหาในช่วงวันทำการของบริษัท จะต้องดำเนินการแก้ไขกลับมาให้ทำงานได้ภายในระยะเวลา 1 ชั่วโมงนับจากแจ้งปัญหา

## Security Requirements
- มีการเก็บประวัติการเข้าใช้งานของผู้ใช้ตาม พ.ร.บ.ธุรกรรมทางอิเล็กทรอนิกส์
- มีการจัดเก็บข้อมูลการเข้าถึงระบบโดยผู้ดูแลระบบ
