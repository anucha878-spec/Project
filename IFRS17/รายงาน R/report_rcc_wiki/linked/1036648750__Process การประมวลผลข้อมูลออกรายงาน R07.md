# Process การประมวลผลข้อมูลออกรายงาน R07

- url: http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1036648750
- http status: 200
- source: Playwright MCP (in-body link, depth 1)

---
**Objective** : ส่งข้อมูล Reconcile actual data and Adjustment posting เพื่อออกรายงาน R07

**Precondition :** เมื่อฝ่ายบัญชีประมวลผล และพิจารณายืนยันข้อมูลการประมวลผลแล้ว ให้ระบบบันทึกข้อมูล Reconcile actual data and Adjustment posting เข้า EDW โดยมีเงื่อนไขดังนี้

**step1 :** ฝ่ายบัญชี กดประมวลผลข้อมูลที่หน้าจอ [EDW-RCC-SD001 หน้าจอประมวลผลข้อมูล และยืนยันการออกรายงาน R](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=930152455)

**step2 : **ฝ่ายบัญชี เลือกข้อมูลเพื่อยืนยันการออกรายงาน R07 เพื่อส่งต่อให้ฝ่ายคณิตศาสตร์นำข้อมูลไปใช้งานต่อ

1. **หาวันที่บันทึกบัญชีจาก Period หน้าจอ**: ระบบนำปี และเดือนที่ได้รับจากหน้าจอ โดยใช้เงื่อนไข วันที่เริ่มต้น และวันที่สิ้นสุดของเดือนที่ต้องการไปหาที่ตาราง [tx_adw_account_head](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_account_head).accounting_date ให้นำ id ที่ได้ไปดำเนินการต่อ
2. **หา Transaction ที่มีการบันทึกบัญชี:** นำข้อมูลจากตาราง [tx_adw_account_head](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_account_head).id ที่ได้จากข้อ 1 ไปหาข้อมูลต่อที่ [tx_adw_account_head_system](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=759529628).tx_adw_account_head_id = [tx_adw_account_head](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_account_head).id and [tx_adw_account_head_system](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=759529628).status = 'POST' and [tx_adw_account_head_system](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=759529628).post_approved_date is not null
3. นำข้อมูลจากตาราง [tx_adw_account_head_system](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=759529628).id ที่ได้จากข้อ 2 ไปหาข้อมูลต่อที่ [tx_adw_transaction_policy](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_transaction_policy).tx_adw_account_head_system_id = [tx_adw_account_head_system](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=759529628).id จากนั้นจะได้กลุ่มของข้อมูลเป็น Transaction ตามวันที่ปิดบัญชีที่ได้จากหน้าจอในข้อ 1 ให้นำข้อมูลที่ได้จากตาราง [tx_adw_transaction_policy](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_transaction_policy) ไปดำเนินการต่อ
4. นำข้อมูลจากตาราง [tx_adw_transaction_policy](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_transaction_policy).id ที่ได้จากข้อ 3 ไปหาข้อมูลต่อที่ [tx_adw_transaction_detail](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_transaction_detail).tx_adw_transaction_policy_id = [tx_adw_transaction_policy](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_transaction_policy).id
5. นำข้อมูล [tx_adw_transaction_detail](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_transaction_detail).id ที่ได้จากข้อ 2 ไปหา**ข้อมูลที่ Model ตามกลุ่ม Claim (model_type = CLAIM)** ที่ [tx_adw_claim_detail](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_claim_detail).tx_adw_transaction_detail_id = [tx_adw_transaction_detail](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_transaction_detail).id
6. นำข้อมูล [tx_adw_claim_detail](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_claim_detail).id ที่ได้จากข้อ 2 ไปหาข้อมูลต่อที่ [tx_adw_double_entry_detail](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_double_entry_detail).tx_adw_claim_detail_id = [tx_adw_claim_detail](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_claim_detail).id
7. เงื่อนไขการดึงข้อมูล ดังนี้ดึงข้อมูลที่เป็นกลุ่ม UL ให้ใช้ Business Line Code ที่เป็น Unit Linked = "07" ([tx_adw_double_entry_detail](http://wiki.thaisamut.co.th/display/RDSADW/tx_adw_double_entry_detail).business_line_code)

2. ระบบจะนำช่วงวันที่ประมวลผล ปี และเดือนที่ได้รับจากหน้าจอ ไปหาที่ตาราง [tx_rcc_reconcile_r07](http://wiki.thaisamut.co.th/display/RDSADW/tx_rcc_reconcile_r07).period และ [tx_rcc_adj_r07](http://wiki.thaisamut.co.th/display/RDSADW/tx_rcc_adj_r07).period
