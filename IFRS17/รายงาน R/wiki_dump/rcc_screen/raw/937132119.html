(Description)

EDW-RCC-SD001 หน้าจอประมวลผลข้อมูล และค้นหาข้อมูล

  1. ระบบแสดงข้อความเริ่มต้น ในส่วนของช่วงประมวลผลข้อมูลเป็น กรุณาระบุ ทั้ง 2 ส่วน

  2. ระบบค้นหาข้อมูลการประมวลผลทั้งหมด มาแสดงในตาราง (กรณีที่เข้าเมนูใหม่ ระบบจะแสดงข้อมูลเพียงแค่ 100 รายการ)

  3. ระบบ Order by ข้อมูลดังนี้

    1. ตามวันที่ เวลา ประมวลผล โดยให้ล่าสุดขึ้นก่อน

    2. ตามสถานะดำเนินการ ให้ลำดับตามนี้
      1. กำลังประมวลผล
      2. รอยืนยันข้อมูล
      3. ยืนยันข้อมูลเรียบร้อย
      4. ประมวลผลไม่สำเร็จ
      5. ยกเลิกการประมวลผล

  4. รูปแบบการแสดงผลของ Date Picker มีรูปแบบเป็น: YYYY/MM กำหนดให้ YYYY เป็นปี ค.ศ. พ.ศ. และ MM เป็นเดือน

    1. เรียงลำดับจาก ปี และ เดือนปัจจุบัน เรียงลำดับจากล่าสุด ไปจนถึงย้อนหลัง
    2. Default เป็น ปีและเดือนปัจจุบัน
    3. การระบุช่วงเพื่อประมวลผลข้อมูล จะต้องไม่เกิน 1 ปี ในการประมวลผลครั้งนั้นๆ (Design ให้รองรับ กรณีที่มีความจำเป็นต้องระบุเกิน 1 ปี ให้แก้ไขได้ง่าย)
    4. การระบุช่วงเพื่อประมวลผลข้อมูล จะต้องไม่สามารถระบุได้เกิน ปี/เดือน ปัจจุบัน
    5. การระบุช่วงเพื่อประมวลผลข้อมูล จะต้องเลือกช่วงปี/เดือน ย้อนหลังได้ไม่เกินปี/เดือนที่ User กำหนด โดยกำหนดให้เป็น Period 2564/06 (Design เป็น config)
  5. ปี ค.ศ. ตั้งแต่ ถึง ให้รูปแบบการแสดงผลของ Label มีรูปแบบเป็น: YYYY/MM กำหนดให้ YYYY เป็นปี ค.ศ. และ MM เป็นเดือน
  6. ปุ่มประมวล ผลสำหรับกดเพื่อประมวลผลโดยดึงข้อมูลตามช่วงประมวลผลข้อมูล และ Generate File CSV วางไว้ที่ Share drive ตามกลุ่มรายงาน R01-R20 และเมื่อประมวลผลสำเร็จให้ระบบจัดเก็บชื่อ Folder ที่ Generate File เพื่อใช้แสดงในหน้าจอ

    1. กรณีกดประมวลผล และพบว่ามีข้อมูลอยู่ในสถานะกำลังประมวลผล จะไม่สามารถกดปุ่ม หรือดำเนินการใดๆ ในหน้าจอได้ เนื่องจากอยู่ระหว่างประมวลผลข้อมูล จนกว่าจะประมวลผลเสร็จสิ้น

      กรณีที่กดประมวลผล และพบว่ามีข้อมูลอยู่ในสถานะรอดำเนินการให้ระบบแสดงข้อความแจ้งเตือน

      1. กรณีที่กดค้นหา และพบว่ามีข้อมูลอยู่ในสถานะกำลังประมวลผลให้ระบบแสดงข้อความแจ้งเตือน

    1. กรณีที่กดค้นหา และพบว่ามีข้อมูลอยู่ในสถานะกำลังประมวลผลให้ระบบแสดงข้อความแจ้งเตือน

  8. ปุ่มล้างเงื่อนไข สำหรับกดเพื่อเคลียร์ค่าทั้งหมด (เหมือนกดปุ่ม Refresh)

  9. รายละเอียดของข้อมูล data grid มีดังต่อไปนี้

    1. รายการประมวลผลข้อมูล จะอิงตาม สถานะดำเนินการ
      1. สถานะเป็น "รอยืนยันข้อมูล" ให้แสดงเครื่องหมาย ใน column แรก และเมื่อกด Icon จะเข้าสู่หน้าจอ EDW-RCC-SD002 หน้าจอแสดงรายละเอียด และยืนยันการออกรายงาน R
    2. ตั้งแต่ ให้แสดง ข้อมูลงวดที่ทำการประมวลผลเริ่มต้น มีรูปแบบเป็น: YYYY/MM กำหนดให้ YYYY เป็นปี ค.ศ. พ.ศ. และ MM เป็นเดือน
    3. ถึง ให้แสดง ข้อมูลงวดที่ทำการประมวลผลสิ้นสุด มีรูปแบบเป็น: YYYY/MM กำหนดให้ YYYY เป็นปี ค.ศ. พ.ศ. และ MM เป็นเดือน
    4. สถานะดำเนินการ มีดังนี้
      1. รอยืนยันข้อมูล หมายถึง เมื่อกดปุ่มประมวลผลแล้ว ข้อมูลครบถ้วนไม่เกิดข้อผิดพลาด เพื่อให้ User สามารถ Download Excel File เพื่อตรวจเช็ค และยืนยันข้อมูลในลำดับถัดไป โดย User สามารถยืนยันข้อมูลได้จากการกด Icon หรือกด เพื่อเข้าสู่ Mode view ข้อมูล ซึ่งระบบจะ Link ไปยังหน้าจอ Link ไปยังหน้าจอ EDW-RCC-SD002 หน้าจอแสดงรายละเอียด และยืนยันการออกรายงาน R
      2. ประมวลผลไม่สำเร็จ หมายถึง เมื่อกดปุ่มประมวลผลแล้ว เกิดข้อผิดพลาดจากระบบ เช่น ระบบไม่สามารถเชื่อมต่อฐานข้อมูลได้
        หรือ Connection lost
      3. ยกเลิกการประมวลผล หมายถึง เมื่อผู้ใช้พบข้อมูลไม่ถูกต้อง และทำการยกเลิกการประมวลผลและยืนยันข้อมูล โดยการระบุที่หน้าจอ EDW-RCC-AM002 แจ้งยกเลิกการประมวลผลและยืนยันข้อมูลออกรายงาน R ตามช่วงการประมวลผล
      4. ยืนยันข้อมูลเรียบร้อย หมายถึง ผู้ใช้ได้ทำการยืนยันข้อมูลทั้งหมด เรียบร้อยแล้วจากหน้าจอ EDW-RCC-SD002 หน้าจอแสดงรายละเอียด และยืนยันการออกรายงาน R เพื่อทำการตรวจสอบและยืนยันข้อมูล
    5. วันที่ เวลา ประมวลผล แสดงวันที่ ที่ผู้ใช้ทำการกดปุ่มประมวลผล มีรูปแบบเป็น: DD/MM/YYYY HH:MM:SS ตัวอย่าง 30/06/2564 13:00:00

      2. กำหนดให้ข้อมูลวันที่เป็น DD เป็นวันที่, YYYY เป็นปี ค.ศ. พ.ศ. และ MM เป็นเดือน
      3. กำหนดให้ข้อมูลเวลาเป็น HH:MM:SS
