Objectives

  • Import File นำเข้าข้อมูล GL Accounting  (Trial Balance) ในรูปแบบไฟล์ CSV เพื่อนำไปประมวลผล  One Page Summary

Process Overview

  • กระบวนการ Import File นำเข้าข้อมูล GL Accounting  (Trial Balance)  เพื่อนำไปประมวลผลออกรายงาน One Page Summary  ของ Reconcile Actual Data & Adjustment Posting (R01-R20)

  • กระทบยอดเงินจาก GL Accounting  (Trial Balance) กับยอดเงินที่ Reconcile Actual Data & Adjustment Posting (R01-R20)

Preconditions

  1. เมื่อมีการกดประมวลผล Reconcile Actual Data & Adjustment Posting EDW-RCC-SD001 หน้าจอประมวลผล และค้นหาข้อมูล
  2. หลังจากระบบประมวลผลรายงาน Reconcile Actual Data & Adjustment Posting (R01-R20) 

 

Process Description

หน้าจอ Pop up นำเข้าข้อมูล GL Accounting  (Trial Balance) และยืนยันการประมวลผล Reconcile Actual Data & Adjustment Posting
