## EDW-RCC-RP010 สำหรับออกรายงาน R10 - Group YRT Premium Receipt PAA ให้ฝ่ายคณิตศาสตร์ ตัวอย่างรายงาน R010 Process การออกรายงานมีเงื่อนไข ดังนี้ 1. สถานะดำเนินการจะต้องเป็น "ประมวลผลสำเร็จ" และผู้ใช้ทำการกดยืนยันข้อมูล จากหน้าจอ [EDW-RCC-SD001 หน้าจอประมวลผลข้อมูล และยืนยันการออกรายงาน R](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=930152455) 2. ระบบนำปี และเดือนที่ได้รับจากหน้าจอ ไปหาที่ตาราง [tx_rcc_output_r10](http://wiki.thaisamut.co.th/display/RDSADW/tx_rcc_output_r10) 3. ทำการ Generate ข้อมูลไปยัง Excel file โดยมีรายละเอียดดังนี้Share drive ให้วางไฟล์ไว้ที่ **$(Default Path)\IFRS17\Report R\XXXXXX_to_XXXXXX_YYYYYYYYYYYYYY** โดยที่ **XXXXXX** แทนข้อมูล Period และ **YYYYYYYYYYYYYY** แทนข้อมูลวันเวลาที่ประมวลผล 4. ชื่อไฟล์เป็น **R10_GroupYRT_PremiumReceipt****_****YYYYMM_to_YYYYMM.csv** 5. ให้ Sorting ตาม Period, PortfolioID 6. ใช้ข้อมูลจากตาราง [tx_rcc_output_r10](http://wiki.thaisamut.co.th/display/RDSADW/tx_rcc_output_r10) เพื่อออกรายงาน R01 ดังนี้ | No | CSV column name | source -
