## EDW-RCC-RP001 สำหรับ Export excel file เพื่อให้ฝ่ายบัญชีตรวจสอบและยืนยันข้อมูลออกรายงาน R11 ตัวอย่างรายงาน R11 ![](http://wiki.thaisamut.co.th/download/attachments/979992897/Greenshot%202565-09-26%2014.57.33.png?version=1&modificationDate=1664179069370&api=v2) Process การออกรายงานมีเงื่อนไขดังนี้ 1. สถานะดำเนินการจะต้องเป็น "ประมวลผลสำเร็จ" หรือ "ยืนยันข้อมูล" เท่านั้นจากหน้าจอ 2. ระบบจะนำช่วงวันที่ประมวลผล ปี และเดือนที่ได้รับจากหน้าจอ ไปหาที่ตาราง [tx_rcc_reconcile_r11](http://wiki.thaisamut.co.th/display/RDSADW/tx_rcc_reconcile_r01).period และ [tx_rcc_adj_r11](http://wiki.thaisamut.co.th/display/RDSADW/tx_rcc_adj_r01).period 3. ทำการ Generate ข้อมูลไปยัง Excel file เพื่อ export ให้ผู้ใช้ โดยตั้งชื่อไฟล์เป็น **R11ACC_YYYYMM_to_YYYYMM.xls** โดยแยกเป็น 2 Sheet มีรายละเอียดดังนี้Sheet ที่ 1 ชื่อ Reconcile Actual Data ใช้ข้อมูลจากตาราง [tx_rcc_reconcile_r11](http://wiki.thaisamut.co.th/display/RDSADW/tx_rcc_reconcile_r01) (Sorting by Account Code, BL) ตารางอธิบายข้อมูลในไฟล์ excel ส่วนของ Reconcile Actual Data | No | csv column name | mapping - [tx_rcc_reconcile_r11](http://wiki.thaisamut.co.th/display/RDSADW/tx_rcc_reconcile_r01) | | --- | --- | --- | | 1 | Period | period | | 2 | Disclosure Type | disclosure_type | | 3 | Account Code | account_code | | 4 | Account Name | account_name | | 5 | BL | business_line | | 6 | Plan Code | plan_code | | 7 | Amount | amount | | 8 | Adj.
