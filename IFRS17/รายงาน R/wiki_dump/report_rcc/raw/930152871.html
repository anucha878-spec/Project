EDW-RCC-RP006 สำหรับ Export excel file เพื่อให้ฝ่ายบัญชีตรวจสอบและยืนยันข้อมูลออกรายงาน R06

ตัวอย่างรายงาน R06 https://docs.google.com/spreadsheets/d/1PjcvE18mHz3kxtcCA77LDTgcQCMvcaSHCwZkCZP6TKU/edit?usp=sharing

Process การออกรายงานมีเงื่อนไขดังนี้

  1. สถานะดำเนินการจะต้องเป็น "ประมวลผลสำเร็จ" หรือ "ยืนยันข้อมูล" ตามสถานะในหน้าจอ EDW-RCC-SD002 หน้าจอแสดงรายละเอียด และยืนยันการออกรายงาน R
  2. ระบบจะนำช่วงวันที่ประมวลผล ปี และเดือนที่ได้รับจากหน้าจอ ไปหาที่ตาราง tx_rcc_reconcile_r06.period และ tx_rcc_adj_r06.period

  3. ทำการ Generate ข้อมูลไปยัง Excel file เพื่อ export ให้ผู้ใช้ โดยตั้งชื่อไฟล์เป็น R06ACC_YYYYMM_to_YYYYMM.xls โดยแยกเป็น 2 Sheet มีรายละเอียดดังนี้

    1. Sheet ที่ 1 ชื่อ Reconcile Actual Data ใช้ข้อมูลจากตาราง tx_rcc_reconcile_r06 (Sorting by Account Code, BL)
