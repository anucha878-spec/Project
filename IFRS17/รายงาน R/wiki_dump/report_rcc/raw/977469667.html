EDW-RCC-RP001 สำหรับ Export excel file เพื่อให้ฝ่ายบัญชีตรวจสอบและยืนยันข้อมูลออกรายงาน R01

ตัวอย่างรายงาน R01

Process การออกรายงานมีเงื่อนไขดังนี้

  1. สถานะดำเนินการจะต้องเป็น "ประมวลผลสำเร็จ" หรือ "ยืนยันข้อมูล" เท่านั้นจากหน้าจอ
  2. ระบบจะนำช่วงวันที่ประมวลผล ปี และเดือนที่ได้รับจากหน้าจอ ไปหาที่ตาราง tx_rcc_reconcile_r01.period และ tx_rcc_adj_r01.period

  3. ทำการ Generate ข้อมูลไปยัง Excel file เพื่อ export ให้ผู้ใช้ โดยตั้งชื่อไฟล์เป็น R01ACC_YYYYMM_to_YYYYMM.xls โดยแยกเป็น 2 Sheet มีรายละเอียดดังนี้

    1. Sheet ที่ 1 ชื่อ Reconcile Actual Data ใช้ข้อมูลจากตาราง tx_rcc_reconcile_r01 (Sorting by Account Code, BL)
