Objectives

  • ประมวลผลออกรายงาน One Page Summary  ของ Reconcile Actual Data & Adjustment Posting (R01-R20)

Process Overview

  • นำข้อมูล GL Accounting  (Trial Balance)  ประมวลผลออกรายงาน One Page Summary  ของ Reconcile Actual Data & Adjustment Posting (R01-R20)

  • กระทบยอดเงินจาก GL Accounting  (Trial Balance) กับยอดเงินที่ Reconcile Actual Data & Adjustment Posting (R01-R20)

Preconditions

  1. เมื่อมีการกดประมวลผล Reconcile Actual Data & Adjustment Posting EDW-RCC-SD001 หน้าจอประมวลผล และค้นหาข้อมูล
  2. กดนำเข้าไฟล์ GL Accounting  (Trial Balance) และยืนยันที่หน้าจอ EDW-ONP-SD001 Pop-up นำเข้าข้อมูล GL Accounting (Trial Balance) และยืนยันการประมวลผล Reconcile Actual Data & Adjustment Posting
  3. หลังจากระบบประมวลผลรายงาน Reconcile Actual Data & Adjustment Posting (R01-R20) 

 

Process Description

 

STEP 1 ขั้นตอนก่อนหน้า
  1. นำเข้าไฟล์ GL Accounting  (Trial Balance) สำเร็จ EDW-ONP-SD001 Pop-up นำเข้าไฟล์ข้อมูล GL Accounting (Trial Balance) และยืนยัน
  2. ประมวลผล Reconcile Actual Data & Adjustment Posting สำเร็จ หรือสำเร็จแต่สถานะ ข้อมูลต้นทางไม่สมบูรณ์ EDW-RCC-SD001 หน้าจอประมวลผล และค้นหาข้อมูล
STEP 2 ประมวลผล Group & Summary ข้อมูล Trial Balance และ Reconcile Data R01 - R20
  1. ประมวลผลข้อมูลตามช่วงประมวลผล Period ตั้งแต่ YYYYMM ถึง YYYYMM
    • นำข้อมูลช่วงประมวลผลปีและเดือนตามหน้าจอเพื่อไปหาข้อมูล ( tx_rcc_monthly_header // period_from >> period_to )
    • อ่านข้อมูลจากไฟล์นำเข้า GL Accounting  (Trial Balance) ที่ระบบจัดเก็บไว้ในช่วงปีและเดือนที่ตรงกัน ( tx_rcc_trial_balance_hd, tx_rcc_trial_balance_dt )

  2. ประมวลผลอ่านยอดเงินตาม Account Code โดยใช้ข้อมูล GL Accounting  (Trial Balance) เป็นหลัก

    • อ่านข้อมูล Initail GL Accounting  (Trial Balance) จากตารางข้อมูล tx_rcc_trial_balance_hd, tx_rcc_trial_balance_dt

      c. บันทึกยอดเงิน Summary R01 - R20 ในตาราง tx_rcc_ops_acc ที่ฟิล์ดข้อมูล sum_rcc_amount ของแต่ละรายงาน R01 ถึง R20

    • กรณีที่ข้อมูล account_code ตาราง tx_rcc_trial_balance_dt ไปอ่านข้อมูล account_code ไม่พบที่ตาราง Reconcile Data R01 - R20

      1. กำหนดให้บันทึกยอดเงิน Summary R01 - R20 ในตาราง tx_rcc_ops_acc ที่ฟิล์ดข้อมูล sum_rcc_amount ของแต่ละรายงาน R01 ถึง R20 เป็นค่าศูนย์ "0.00"
