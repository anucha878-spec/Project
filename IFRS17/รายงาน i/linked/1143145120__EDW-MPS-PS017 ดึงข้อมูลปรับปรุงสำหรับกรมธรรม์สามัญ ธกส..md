# EDW-MPS-PS017 ดึงข้อมูลปรับปรุงสำหรับกรมธรรม์สามัญ ธกส.

| **No.** | **Topic** | **Description** |
| --- | --- | --- |
| 1 | วัตถุประสงค์(Objective) | จัดเตรียมข้อมูล Monthly Policy Status Snapshot สำหรับข้อมูลปรับปรุงสำหรับกรมธรรม์สามัญ ธกส. เพื่อใช้สำหรับการตั้งฐานข้อมูล, การนำเข้าข้อมูลกรมธรรม์เคสใหม่ หรือ Reinstate, และการ Update ข้อมูลกรมธรรม์ที่เคยนำเข้าในฐานข้อมูล Monthly Policy Status Snapshot |
| 2 | สัมพันธ์กับกระบวนการ(Link to process) | |
| 3 | เวลาประมวลผล (Time) | ทุกวันที่ 1 - 10 ของทุกเดือน เวลา 05.00 น. (ตี 5) |
| 4 | ข้อมูลตั้งต้น(Input) | ข้อมูลปรับปรุงสำหรับกรมธรรม์สามัญ ธกส.จากฐานข้อมูลในระบบ AS400 |
| 5 | ข้อมูลที่ได้จากระบบ(Output) | ข้อมูลกรมธรรม์ของ MPS ที่ตาราง [tx_mps_backdate_ord](http://wiki.thaisamut.co.th/display/RDSADW/tx_mps_backdate_ord)[tx_mps_backdate_rider](http://wiki.thaisamut.co.th/display/RDSADW/tx_mps_backdate_rider) |
| 6 | อธิบายรายละเอียด(Description) | เมื่อถึงเวลาระบบจะตรวจสอบข้อมูลตามขั้นตอน ดังนี้ตรวจสอบข้อมูลที่ [tx_mps_backdate_hd](http://wiki.thaisamut.co.th/display/RDSADW/tx_mps_backdate_hd) ด้วยเงื่อนไขของ period_year = ปีของรอบประมวลผล และ period_month = เดือนของรอบประมวลผล และ ms_import_status_id = 7 ถ้าพบข้อมูลให้จบการทำงานทันทีกรณีที่ ข้อ 1 ไม่พบข้อมูล ให้ดึงข้อมูลจาก [WS_EDW3_26 ตรวจสอบสถานะพร้อมดึงข้อมูลปรับปรุงสำหรับกรมธรรม์สามัญ ธกส.](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=1143145090) โดย**กรณีที่ข้อมูลที่ได้เป็น "EMPTY"** จะจบขั้นตอนทันที่ ระบบจะลงข้อมูลสถานะ "รอนำเข้าข้อมูล" จำนวนรายการ เป็น 0 และหมายเหตุ "ไม่พบข้อมูลนำเข้าประจำเดือน"**กรณีที่ข้อมูลที่ได้เป็น "NODATA"** ระบบจะลงข้อมูลสถานะ "ยืนยันนำเข้าข้อมูลสำเร็จ" จำนวนรายการ เป็น 0 และหมายเหตุ "ไม่มีข้อมูลนำเข้าประจำเดือน"**กรณีที่ข้อมูลที่ได้เป็น "COMPLETE"** ระบบจะทำการตามกระบวนการต่อไปนี้ดึงข้อมูลจาก [WS_EDW3_05 ดึงข้อมูลกรมธรรม์หลักสามัญ สำหรับธกส.backdate](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=940736574) **กรณีที่ไม่พบข้อมูล** ระบบจะลงข้อมูลสถานะ "นำเข้าไม่สำเร็จ" จำนวนรายการ เป็น 0 และหมายเหตุ "ไม่พบรายการข้อมูลนำเข้า กรุณาติดต่อทีม AS400"**กรณีที่พบข้อมูล** ระบบจะลงข้อมูลสถานะ "ยืนยันนำเข้าข้อมูลสำเร็จ" จำนวนรายการ ตามจำนวนรายการข้อมูลที่พบ และบันทึกข้อมูลง [tx_mps_backdate_ord](http://wiki.thaisamut.co.th/display/RDSADW/tx_mps_backdate_ord)ดึงข้อมูลจาก [WS_EDW3_16 ดึงข้อมูลกรมธรรม์สามัญ ธกส. ส่วนสัญญาเพิ่มเติม](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=965149004)**กรณีที่พบข้อมูล** ให้บันทึกข้อมูลง [tx_mps_backdate_rider](http://wiki.thaisamut.co.th/display/RDSADW/tx_mps_backdate_rider) |
