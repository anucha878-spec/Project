---
กระบวนการ “นำส่งไฟล์เพื่อติด Tagging” ในหน้า [EDW-MPS-SD008 หน้าจอประมวลผล MPS](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=918028352) สำหรับการ Generate และ Export รายงาน I01-I05 สำหรับกรมธรรม์เคสใหม่ และกรมธรรม์ที่ยังไม่ติด Tagging ไปวางไว้ที่ Landing Path ที่กำหนด

StateDiagram: [state_diagram_genUntag](http://wiki.thaisamut.co.th/download/attachments/932544695/Untitled%20Diagram-Page-1.drawio%20%281%29.png?version=1&modificationDate=1646191992668&api=v2)

>
**Process Overview**

1. หากรมธรรม์เคสใหม่, เคสใหม่ที่มีการ Freelook, หรือ Reinstate ในรอบการประมวลผลเดือนปัจจุบัน
2. Generate ข้อมูลกรมธรรม์จากข้อ 1. ลงตาราง Untag แยกตามประเภทกรมธรรม์
3. Generate ข้อมูลกรมธรรม์จากข้อ 2. ออกมาในรูปแบบไฟล์ .csv
4. บันทึก Log
