| รอนำเข้าข้อมูล | ระบบอยู่ในระหว่างกระบวนการเตรียมข้อมูลสิ้นเดือน ระบบ Disable ทุกปุ่ม ดังนี้“รายละเอียด”“ยกเลิกรายการ”“นำส่งไฟล์เพื่อติด Tagging” เมื่อสถานะของกระบวนการเตรียมข้อมูลสิ้นเดือน ทั้งหมดไม่เป็น "ยืนยันนำเข้าข้อมูลสำเร็จ" (header ของหน้าจอ และ batch)“นำเข้าไฟล์ Tagging”“ยืนยันส่งข้อมูลเข้าระบบ EDW”ระบบจะ Enable ปุ่ม “นำส่งไฟล์เพื่อติด Tagging” เมื่อสถานะของกระบวนการเตรียมข้อมูลสิ้นเดือนทั้งหมดเป็น "ยืนยันนำเข้าข้อมูลสำเร็จ" (header ของหน้าจอ และ batch) |
| กำลังดำเนินการส่งไฟล์ I01-I05 เพื่อติด Tagging | เกิดจากการกดปุ่ม “นำส่งไฟล์เพื่อติด Tagging”ระบบกำลังดำเนินการนำส่งไฟล์เพื่อขอ Taggingระบบ Disable ทุกปุ่ม ดังนี้“รายละเอียด”“ยกเลิกรายการ”“นำส่งไฟล์เพื่อติด Tagging”“นำเข้าไฟล์ Tagging”“ส่งข้อมูลเข้าระบบ EDW” |
| นำส่งไฟล์ I01-I05 เพื่อติด Tagging ไม่สำเร็จ | ระบบดำเนินการนำส่งไฟล์ไม่สำเร็จระบบ Enable ปุ่ม ดังนี้“รายละเอียด”“ยกเลิกรายการ”“นำส่งไฟล์เพื่อติด Tagging”หน้าจอ popup รายละเอียด ส่วนแสดงผล “รายการข้อมูล I01-I05 เพื่อติด Tagging” ระบบแสดง “สถานะข้อมูล” ด้วยตัวอักษรสีแดง ระบบไม่แสดง “จำนวนรายการ”ระบบส่ง Automail แจ้งผู้เกี่ยวข้อง (อ้างอิง Email Template จาก Appendix [A6.6 Email แจ้งผลการนำส่งไฟล์ I01-I05 เพื่อติด Tagging](https://docs.google.com/document/d/1icFu-0Zyw472qLpycqeMJ1ZqkVnnaYeOxVWVgTzFPz8/edit#heading=h.d2348pym95x4)) |
| รอนำเข้าไฟล์ Tagging | ระบบนำส่งไฟล์เพื่อติด Tagging สำเร็จระบบ Enable ปุ่ม ดังนี้“รายละเอียด”“ยกเลิกรายการ”“นำเข้าไฟล์ Tagging”หน้าจอ popup รายละเอียด ส่วนแสดงผล “รายการข้อมูล I01-I05 เพื่อติด Tagging” ระบบแสดง “สถานะข้อมูล” และ “จำนวนรายการ”ระบบส่ง Automail แจ้งผู้เกี่ยวข้อง (อ้างอิง Email Template จาก Appendix [A6.6 Email แจ้งผลการนำส่งไฟล์ I01-I05 เพื่อติด Tagging](https://docs.google.com/document/d/1icFu-0Zyw472qLpycqeMJ1ZqkVnnaYeOxVWVgTzFPz8/edit#heading=h.d2348pym95x4)) |
| กำลังดำเนินการนำเข้าไฟล์ Tagging | เกิดจากการกดปุ่ม “นำเข้าไฟล์ Tagging”ระบบกำลังดำเนินการนำเข้าไฟล์ Taggingระบบ Enable ปุ่ม “รายละเอียด” |
| นำเข้าไฟล์ Tagging ไม่สำเร็จ | ระบบตรวจสอบไฟล์ต้นทางไม่ถูกต้อง หรือระบบดำเนินการ Update ติด Tagging ไม่สำเร็จระบบ Enable ปุ่ม ดังนี้“รายละเอียด”“ยกเลิกรายการ”“นำเข้าไฟล์ Tagging”หน้าจอ popup รายละเอียด ส่วนแสดงผล “รายการข้อมูลนำเข้า”ระบบแสดง “สถานะข้อมูล” ด้วยตัวอักษรสีแดง, แสดง “หมายเหตุ”กรณีพบข้อมูลในไฟล์นำเข้า Invalid Data Format ระบบแสดง icon “Error File” ระบบไม่แสดง “จำนวนรายการ”ระบบส่ง Automail แจ้งผู้เกี่ยวข้อง (อ้างอิง Email Template จาก Appendix [A6.7 Email แจ้งผล Update ICG/RCG Tagging](https://docs.google.com/document/d/1icFu-0Zyw472qLpycqeMJ1ZqkVnnaYeOxVWVgTzFPz8/edit#heading=h.sq7djgqc5hby)) |
| รอยืนยันเพื่อส่งข้อมูลเข้าระบบ EDW | ระบบดำเนินการนำเข้าไฟล์ Tagging สำเร็จระบบ Enable ปุ่ม ดังนี้“รายละเอียด”“ยกเลิกรายการ”“ยืนยันส่งข้อมูลเข้าระบบ EDW”หน้าจอ popup รายละเอียด ส่วนแสดงผล “รายการข้อมูลนำเข้า”ระบบแสดง “จำนวนรายการ”, “สถานะข้อมูล” ระบบไม่แสดง “หมายเหตุ” และ icon “Error File” ระบบส่ง Automail แจ้งผู้เกี่ยวข้อง (อ้างอิง Email Template จาก Appendix [A6.7 Email แจ้งผล Update ICG/RCG Tagging](https://docs.google.com/document/d/1icFu-0Zyw472qLpycqeMJ1ZqkVnnaYeOxVWVgTzFPz8/edit#heading=h.sq7djgqc5hby)) |
| กำลังดำเนินส่งข้อมูลเข้าระบบ EDW | เกิดจากการกดปุ่ม “ยืนยันส่งข้อมูลเข้าระบบ EDW”ระบบกำลังดำเนินการส่งข้อมูลเข้าระบบ EDWระบบ Enable ปุ่ม “รายละเอียด” |
| ระบบส่งข้อมูลเข้าระบบ EDW ไม่สำเร็จ | ระบบดำเนินการส่งข้อมูลเข้าระบบ EDW ไม่สำเร็จระบบ Enable ปุ่ม ดังนี้“รายละเอียด”“ยกเลิกรายการ”“ยืนยันส่งข้อมูลเข้าระบบ EDW”หน้าจอ popup รายละเอียด ส่วนแสดงผล “รายการข้อมูล output”ระบบแสดง “สถานะข้อมูล” ด้วยตัวอักษรสีแดง และ “หมายเหตุ”ระบบไม่แสดง “จำนวนรายการ”ระบบส่ง Automail แจ้งผู้ที่เกี่ยวข้อง (อ้างอิง Email Template จาก Appendix [A6.8 Email แจ้งผลการส่งข้อมูลเข้าระบบ EDW](https://docs.google.com/document/d/1icFu-0Zyw472qLpycqeMJ1ZqkVnnaYeOxVWVgTzFPz8/edit#heading=h.4dp9r8eh42hf)) |
| ระบบส่งข้อมูลเข้าระบบ EDW สำเร็จ | ระบบดำเนินการส่งข้อมูลเข้าระบบ EDW สำเร็จระบบ Enable ปุ่ม ดังนี้“รายละเอียด”“ยกเลิกรายการ”“ยืนยันส่งข้อมูลเข้าระบบ EDW”หน้าจอ popup รายละเอียด ส่วนแสดงผล “รายการข้อมูล output”ระบบแสดง “จำนวนรายการ” และ “สถานะข้อมูล”ระบบไม่แสดง “หมายเหตุ”ระบบส่ง Automail แจ้งผู้ที่เกี่ยวข้อง (อ้างอิง Email Template จาก Appendix [A6.8 Email แจ้งผลการส่งข้อมูลเข้าระบบ EDW](https://docs.google.com/document/d/1icFu-0Zyw472qLpycqeMJ1ZqkVnnaYeOxVWVgTzFPz8/edit#heading=h.4dp9r8eh42hf)) |
| ยกเลิกรายการ | เกิดจากการกดปุ่ม “ยกเลิกรายการ”ระบบ Enable ปุ่มดังนี้“รายละเอียด”“นำส่งไฟล์เพื่อติด Tagging” เมื่อสถานะของกระบวนการเตรียมข้อมูลสิ้นเดือนทั้งหมดเป็น "ยืนยันนำเข้าข้อมูลสำเร็จ" ทั้งหมด (header ของหน้าจอ และ batch) |
| Backup ไม่สำเร็จ | เกิดจาก กดปุ่ม 'ส่งข้อมูลเข้าระบบ EDW' และ insert ข้อมูล backup ไม่สำเร็จ (delete ข้อมูลเก่าใน table และทำการ insert ใหม่)ระบบ Enable ปุ่มดังนี้ 'ยกเลิกรายการ''ส่งข้อมูลเข้าระบบ EDW'ระบบส่ง Automail แจ้งผู้ที่เกี่ยวข้อง (อ้างอิง Email Template จาก Appendix [A6.8 Email แจ้งผลการส่งข้อมูลเข้าระบบ EDW](https://docs.google.com/document/d/1icFu-0Zyw472qLpycqeMJ1ZqkVnnaYeOxVWVgTzFPz8/edit#heading=h.4dp9r8eh42hf)) |
