##### Input File

| ชื่อ Column | Description | ตัวอย่าง | Nullable | Size |
| --- | --- | --- | --- | --- |
| Period | รอบการประมวลผล format YYYYMMYYYY = ปี 4 Digits (ค.ศ.)MM = เดือน 2 Digits | 202209 | N | 6 |
| PolicyNumber | เลขที่กรมธรรม์ | ข0000001 | N | 50 |
| ProphetPlanCode | แบบประกันในรูปแบบ prophet | OC98 | N | 50 |
| RIPortfolioName | IFRS17 RI Portfolio name | ORD_MRPORD_MRNINR_CIPINR_CININR_MENSPA_ADN | N | 50 |
| RICohortYear | ปี | 2022 | N | 10 |
| RIProfitabilityGroup | RI Profitability group | RING = Reinsurance Net gainRINC = Reinsurance Net CostRIRE = Reinsurance Remaining | N | 10 |
| RIPortfolioID | Reinsruance contract group(RCG) | ORD_MRN-2022-RING | N | 50 |
