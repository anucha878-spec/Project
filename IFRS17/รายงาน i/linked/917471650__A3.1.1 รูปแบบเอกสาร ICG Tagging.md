##### Input File

| ชื่อ Column | Description | ตัวอย่าง | Nullable | Size |
| --- | --- | --- | --- | --- |
| Period | รอบการประมวลผล ในรูปแบบ YYYYMMYYYY = ปี 4 Digits (ค.ศ.)MM = เดือน 2 Digits | 202208 | N | 6 |
| PolicyNumber | เลขที่กรมธรรม์ | ข0000001 | N | 50 |
| ProphetPlanCode | แบบประกันในรูปแบบ prophet | OC98 | N | 50 |
| PortfolioName | IFRS17 Portfolio name | IND_END = สะสมทรัพย์อุตสาหกรรมORD_ANN = บำนาญสามัญORD_END = สะสมทรัพย์สามัญORD_PAR = PAR สามัญORD_SPA = Single Premium สามัญORD_TRM = Term สามัญORD_WOL = ตลอดชีพ สามัญUNT_LNK = ยูนิตลิงค์ | N | 50 |
| CohortYear | ปีที่เริ่มสัญญา | 2022 | N | 10 |
| ProfitabilityGroup | Profitability group | NSPO = OnerousONRS = Remaining contractREMC = No significant possibiliety | N | 10 |
| PortfolioID | Insurance contract group(ICG) | ORD_END-2022-ONRS | N | 50 |
