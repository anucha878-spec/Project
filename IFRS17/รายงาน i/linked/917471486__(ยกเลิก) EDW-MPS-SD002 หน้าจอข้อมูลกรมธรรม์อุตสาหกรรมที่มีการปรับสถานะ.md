หากดำเนินการเสร็จสิ้น ระบบดำเนินการดังนี้

1. กรณีสำเร็จ ปรับสถานะการนำเข้าข้อมูลเป็น** “รอยืนยันข้อมูล”**
2. กรณีไม่สำเร็จ ปรับสถานะการนำเข้าข้อมูล เป็น** "นำเข้าไม่สำเร็จ" **พร้อมแจ้ง Error Message ที่ “หมายเหตุ”

รายละเอียดของ Data Grid ดังนี้ ดึงข้อมูลจาก [tx_mps_backdate_ind_hd](http://wiki.thaisamut.co.th/display/RDSADW/tx_mps_backdate_ind_hd)

1. “ครั้งที่” แสดงจำนวนรายการนำเข้าข้อมูลในรอบการประมวลผลปัจจุบัน
2. “ดำเนินการ” ภายใต้ column มีปุ่มการทำงาน ดังนี้
3. ปุ่ม** "ยืนยันนำเข้าข้อมูล"** สำหรับการยืนยันนำข้อมูลเข้าระบบ EDW หากกดปุ่มระบบดำเนินการเปิด Popup Screen EDW-MPS-SD002_2 เพื่อยืนยันนำเข้าข้อมูล รายละเอียดดังนี้เงื่อนไขการ Enable ปุ่มเมื่อมีสถานะการนำเข้าข้อมูลเป็น** “รอยืนยันข้อมูล” **

หากยืนยันนำเข้าข้อมูลสำเร็จ ปรับสถานะการนำเข้าข้อมูลเป็น** “ยืนยันนำเข้าข้อมูลสำเร็จ”**

หากยืนยันนำเข้าข้อมูลไม่สำเร็จ ไม่ปรับสถานะสถานะการนำเข้าข้อมูล

ปุ่ม** “ยกเลิก” **สำหรับยกเลิกข้อมูลนำเข้า หากกดปุ่ม ระบบจะดำเนินการเปิด Popup Screen EDW-MPS-SD002_3 หน้าจอ popup ยกเลิก เพื่อยืนยันการยกเลิกข้อมูล พร้อมระบุหมายเหตุ โดยมีรายละเอียด ดังนี้

1. เงื่อนไขการ Enable ปุ่ม
2. เมื่อยังอยู่ในกระบวนการนำเข้าข้อมูล (สถานะการประมวลผล MPS จาก [หน้าจอประมวลผล MPS](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=918028352) เป็น “รอนำเข้าข้อมูล” หรือ “ยกเลิกรายการ”) และสถานะการนำเข้าข้อมูลเป็น **"รอยืนยันข้อมูล" **หรือ **"ยืนยันนำเข้าข้อมูลสำเร็จ”**

หลังจากดำเนินการเสร็จสิ้น

1. หากยกเลิกสำเร็จ ปรับสถานะการนำเข้าข้อมูลเป็น **“ยกเลิกข้อมูล”**
2. หากยกเลิกไม่สำเร็จ ไม่ปรับสถานะรายการการนำเข้าข้อมูล

“วัน และเวลาดำเนินการ” แสดงวัน และเวลานำเข้าข้อมูล ในรูปแบบ DD/MM/YYYY HH:MM:SS รายละเอียด ดังนี้

1. DD คือ วันที่แบบ 01-31
2. MM คือ เดือนแบบ 01-12
3. YYYY คือ ปี **พ.ศ. แบบ 2500-9999**
4. HH คือ ชั่วโมง แบบ 00-24
5. MM คือ นาที แบบ 00-60
6. SS คือ วินาที แบบ 00-60

“ผู้ดำเนินการ” แสดงผู้นำเข้าข้อมูล ในรูปแบบ Login ID

“สถานะ” แสดงสถานะการนำเข้าข้อมูล (อ้างอิง สถานะ จาก Appendix [A10.2 สถานะรายการข้อมูลหน้าจอ ข้อมูลกรมธรรม์อุตสาหกรรมที่มีการปรับสถานะ](http://wiki.thaisamut.co.th/pages/viewpage.action?pageId=917471561))

“รายการปรับปรุง” แสดงจำนวนรายการกรมธรรม์อุตสาหกรรมที่มีการปรับสถานะกรมธรรม์สิ้นผล

“หมายเหตุ” แสดงหมายเหตุที่ระบุจากการ Update ข้อมูลเข้าระบบ, การยืนยันข้อมูลนำเข้า, การยกเลิกข้อมูลนำเข้า, หรือ Error Message จากระบบกรณีพบ Invalid Data Format

"Download" คือปุ่มสำหรับ download ไฟล์ (.xlsx) เพื่อตรวจสอบรายละเอียดของข้อมูลที่มีการปรับสถานะก่อนที่จะยืนยันการนำเข้าข้อมูล โดยดึงข้อมูลจาก table [tx_mps_backdate_ind](http://wiki.thaisamut.co.th/display/RDSADW/tx_mps_backdate_ind)
